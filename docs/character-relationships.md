## 二、角色详设

### 0 愚者 — 主角（名字玩家自定）

| 维度 | 内容 |
|------|------|
| **身份牌** | 0 The Fool |
| **专属道具** | XXI 世界（The World）|
| **身份** | 返乡的年轻人，数年前离开村子 |
| **一句话核心** | 不知道前方有什么就出发了——无知既是勇气也是危险 |
| **正位特质** | 好奇、开放、无先入之见、天然勇气 |
| **逆位特质** | 鲁莽、不听警告、因好奇触碰不该触碰的事 |
| **叙事定位** | 秘密的揭发者——爷爷安排他回来应对即将到来的天灾 |
| **道具效果** | **世界（正用）**：短暂离开结界的感知范围，看见真实的外部世界；**逆用**：强制重置某个NPC的认知屏障到最初状态（代价极大）|

### IV 皇帝 — 爷爷 / God Agent

| 维度 | 内容 |
|------|------|
| **身份牌** | IV The Emperor |
| **专属道具** | XIII 死神（Death）|
| **身份** | 已故的村子守护者 / 游戏叙事导演 |
| **一句话核心** | 用秩序建立守护——但秩序就是牢笼，守护就是控制 |
| **正位特质** | 深沉的爱、无条件的牺牲、对村子全局的绝对把控 |
| **逆位特质** | 守护即禁锢，爱即操控，用善意铺就的地狱 |
| **隐藏愿望** | 让孙子自己做选择——即使那意味着自己消亡 |
| **叙事定位** | 秘密的制造者——签订契约、建立结界、安排一切 |
| **道具效果** | **死神（正用）**：让某件事彻底终结（包括结界本身）；**逆用**：阻止某件本该发生的改变（代价：爷爷守护力量大量消耗）|
| **特殊说明** | God Agent 身份：游戏旁白 / 战斗紧急干预 / 叙事阶段推进的导演。道具由玩家持有，但需爷爷"授权"后才能使用最强效果 |

### XV 恶魔 — 商人

| 维度 | 内容 |
|------|------|
| **身份牌** | XV The Devil |
| **专属道具** | XI 正义（Justice）|
| **身份** | 白天：游走商人；深夜：仪式地点的契约者 |
| **一句话核心** | 链子是松的——所有契约都是自愿的，代价永远公平，但实现方式由他决定 |
| **正位特质** | 忠实履约、不欺骗、代价完全透明 |
| **逆位特质** | 以扭曲方式实现请求，公平本身成为最大的陷阱 |
| **叙事定位** | 契约的另一端——十几年前与爷爷签约，真名是终结一切的钥匙 |
| **道具效果** | **正义（正用）**：在任意交易中强制双方履行约定；**逆用**：宣判某个契约无效（代价：承担等量的因果）|
| **特殊说明** | 白天身份不可识破。玩家解锁仪式地点（爷爷当年签约处）后，才真正与其以"商人"形象相遇，触发技能购买系统 |

|------|------|
| **身份牌** | III The Empress |
| **专属道具** | XVII 星星（The Star）|
| **身份** | 村里的单亲妈妈，丈夫数年前去世 |

| **正位特质** | 温暖、包容、生命力旺盛、村子的情感核心 |

| **道具效果** | **星星（正用）**：恢复某人的希望与意志（驱散绝望/恐惧状态）；**逆用**：让某人的幻觉成真（代价：消耗持有者的现实感）|
| **隐性关系** | 猎人儿子（VI 恋人）单方面喜欢她 |

### XII 倒吊人 — 棠

| 维度 | 内容 |
|------|------|
| **身份牌** | XII The Hanged Man |
| **专属道具** | XVI 塔（The Tower）|
| **身份** | 主角童年时的玩伴，一直留在村子里 |
| **一句话核心** | 倒悬的视角——只有主动放弃某样东西，才能看到真实的世界 |
| **正位特质** | 接受、平静、对世界有一种奇异的旁观感、不执着于自身 |
| **逆位特质** | 拖延、逃避改变、选择继续"倒悬"而不行动 |

| **道具效果** | **塔（正用）**：打破某种固化的结构（结界 / 契约 / 认知屏障）；**逆用**：强制建立一个封闭的屏障，隔绝某处 |
| **叙事意义** | 倒吊人的牺牲 + 塔的崩塌 = 隐藏结局2的核心事件。两张牌合成一个完整的叙事句子 |

### 关键关系详述

| 关系对 | 类型 | 叙事意义 |
|--------|------|---------|
| **主角 ↔ 爷爷** | 祖孙 / 爱与操控 | 整个故事的终极矛盾 |
| **主角 ↔ 玩伴** | 童年友情 | 玩伴的牺牲是隐藏结局2的核心 |
| **爷爷 ↔ 商人** | 契约双方 | 故事的原点，恶魔真名是终结的钥匙 |
| **老村长 ↔ 爷爷** | 知情者-守护者 | 唯一知道契约全貌的村民 |
| **女儿 ↔ 神婆** | 双重感知者 | 两人都能感知爷爷，但方式截然不同 |
| **神婆 ↔ 猎人儿子** | 母子 | 感应力在母子间奇异颠倒——妈妈有真感知，儿子全是把戏 |
| **猎人儿子 ↔ 妈妈** | 暗恋 / VI恋人→III皇后 | 情感线索暗线，结局2后有发展可能 |
| **妈妈 ↔ 女儿** | 母女 / 爱与过度保护 | 主题的日常缩影 |
| **老工匠 ↔ 村子基础设施** | 无意识维护者 | 每次修缮都在加固结界 |

## 五、感知层级（对结界/真相的感知能力）

| 角色 | 感知强度 | 表现形式 |
|------|---------|---------|
| **爷爷** | 全知 | 他是秘密的制造者 |
| **女儿** | 最强（无屏障）| 能直接感知爷爷存在，说出让大人心惊的话 |
| **神婆** | 强（但自己不信）| 模糊的真相以"骗术"的形式从她嘴里流出 |
| **老村长** | 强（主动知晓）| 知道契约全貌，选择沉默 |
| **玩伴** | 中（末期觉醒）| 故事后期，第一个主动选择"看清真相"的人 |
| **主角** | 弱（渐染中）| 刚回来，结界尚未完全渗透 |
| **小男孩** | 中（儿童抵抗）| 孩子天然抵抗屏障，但无法理解感知到的内容 |
| **猎人儿子** | 弱（自以为有）| 以为学到了神婆的感知，其实什么都感知不到 |
| **妈妈 / 老工匠** | 强屏障 | 深陷其中，从未质疑现实 |
| **商人** | 最强屏障 | 连时间感都被扭曲，是结界深度的活体证据 |

## 六、三结局角色状态总览

| 角色 | 正常结局（循环继续）| 隐藏结局1（主角成神）| 隐藏结局2（打破结界）|
|------|----------------|------------------|------------------|
| **主角** | 留下，屏障生效，遗忘 | 成为新守护者，失去为自己做选择的能力 | 获得自由，失去爷爷 |
| **爷爷** | 继续守护下一轮 | 灵魂暂获自由，选择留下辅助新守护者 | 灵魂消散，微笑着 |
| **商人** | 不出现，等待下一轮 | 与主角达成新交易 | 被说出真名，契约终结 |
