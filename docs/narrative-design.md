## 1. 核心哲学与世界观

### 1.1 守护本身就是禁锢

> "真正的恶魔是'我为你好'这句话。"

这不是善恶对立的故事。这是一个关于**爱的悖论**的故事：

- 用爱建造笼子，锁门的人是最爱你的人
- 保护者把被保护者当作所有物，替他们做决定
- 守护者渴望他们永远安全——但安全的代价是自由
- 真正的自由也许意味着失去最爱的人

世界里没有纯粹的善恶。每个角色都有一体两面——白天的温暖是真的，夜晚的阴影也是真的。连恶魔都不是坏人。

### 1.2 恶魔哲学

恶魔不是邪恶的存在，只是**做交易的存在**。

- **正位（利他）**：满足他人的愿望，帮人实现渴望——但代价从不免费
- **逆位（利己）**：操纵欲望，让人沉溺于交易无法自拔

"恶魔"这个标签是人类在后悔时贴给交易对象的。你签了合同、拿了好处、付了代价，然后说对方是恶魔——但你当时是自愿的。

**爷爷签下契约时，恶魔是正位的**——他真的帮爷爷实现了守护的愿望。代价也说得清清楚楚。问题不在恶魔，问题在于"守护"这个愿望本身就有毒。

### 1.3 涌现式叙事设计哲学

**目标：不做线性剧情，做涌现式叙事。**

NPC 有连贯的人格记忆，危机/事件由 God Agent 动态生成。每次重开都有新故事，但每个 NPC 都像真实的人。当玩家最终发现"上帝就是爷爷"时，回头看所有曾经觉得奇怪的对话选项——每一个都是爷爷在说"我在这里，我一直在保护你们"。

## 2. 世界结构：三层洋葱

### 第一层：表层——温馨田园

玩家最初看到的一切：

- 一个与世隔绝但宁静美好的小村庄
- 热情的邻居、温暖的日常、种田采矿的田园生活
- 上帝只是天气系统和旁白的提供者
- 一切看起来像星露谷物语

**设计意图**：让玩家真正爱上这个村子，建立深厚的情感连接。只有先爱上，之后的揭示才有冲击力。

### 第二层：中层——结界与认知屏障

逐渐浮现的异常：

- 没有人能离开村子，也没有外人能进入
- 村民对此毫无察觉——认知屏障模糊了所有人关于"外面"的认知
- 老井传出低语、红雪降临、边界出现裂纹光
- NPC 偶尔说出让人心惊的话但自己也不明白

**结界的三层代价**：
1. **灵魂代价**：爷爷的灵魂永远绑定在村子，成为守护者但失去自由
2. **空间代价**：结界封锁村子，没有人能出入
3. **认知代价**：所有村民的感知被模糊化，无法意识到自己被困

### 第三层：深层——交易本质

最终揭示的真相：

- 村子是一个用爱构建的牢笼
- 上帝不是神，是一个被困的老人——玩家的爷爷
- 守护仪式、法阵组件、编织图案……一切"传统"都是契约的维护手段
- 恶魔只是交易的中介，真正的"恶魔"是"我为你好"的逻辑
- 每一层保护都是一层禁锢

## 3. 塔罗象征系统

### 3.1 为什么选塔罗大阿卡那

- 大阿卡那22张牌本身就是一个完整的"愚者之旅"——从天真到觉醒的灵魂成长
- 正位/逆位天然适配"白天面/阴影面"的一体两面设计
- 每张牌有丰富的视觉象征，为角色外观、道具、场景提供原生意象
- 不使用七宗罪——七宗罪是道德审判框架，与"没有纯粹善恶"的世界观冲突

### 3.2 选定的11张牌

从22张大阿卡那中选取11张，构成一个完整的叙事弧：

| 编号 | 牌名 | 角色 | 选择理由 |
|------|------|------|---------|
| **0** | **愚者 The Fool** | 主人公 | 愚者之旅的起点。天真、勇气、不知道前方有什么就出发了。回到村子就是一次新的愚者之旅 |
| **II** | **女祭司 The High Priestess** | 陈婆 | 帷幕两侧的守门人。她坐在记忆与遗忘之间，潜意识的知识通过编织流淌出来 |
| **VIII** | **力量 Strength** | Bob | 不是蛮力，是驯服内心野兽的力量。Bob 压制着恐惧和悲伤，正位是坚韧，逆位是被恐惧吞噬 |
| **IX** | **隐者 The Hermit** | Dave | 提灯的隐士，用谜语照亮道路。知道全部真相但只能用隐晦的方式传递 |
| **XII** | **倒吊人 The Hanged Man** | 爷爷（God Agent） | 自愿倒悬、牺牲自我换取新视角。爷爷自愿献出灵魂，被倒悬在守护与禁锢之间 |
| **XIV** | **节制 Temperance** | Carol | 两杯之间倒水的天使——平衡、中和、维持秩序。Carol 无意识地维护着仪式的平衡 |
| **XV** | **恶魔 The Devil** | 恶魔 | 锁链与诱惑。但牌面上被锁的人脖子上的链子是松的——他们随时可以取下来。契约是自愿的 |
| **XVI** | **塔 The Tower** | Marco | 雷电击中高塔，人从窗口坠落。既有的信念结构被摧毁 |
| **XVII** | **星星 The Star** | Lily | 塔之后的希望。纯粹的信念、清澈的感知。她是废墟中仰望星空的人 |
| **XVIII** | **月亮 The Moon** | Alice | 月光下的小径，充满幻觉和直觉。Alice 隐约感受到真相但看不清，地图碎片就是月下朦胧的路 |
| **XIX** | **太阳 The Sun** | Erik | 炉火即是太阳。创造、温暖、生命力。他锻造的不只是铁器，是维持世界运转的组件 |

### 3.3 正位与逆位的运作

每个角色的正位是**白天面**——日常生活中展现的那一面，温暖、功能性的、社会化的。

逆位是**阴影面**——夜晚浮现的那一面，恐惧、渴望、被压抑的真相。

正逆位不是善恶之分，是同一枚硬币的两面：

- Bob（力量）的正位是坚韧可靠，逆位是被恐惧控制逃避真相——两者都是"力量"的表现
- Carol（节制）的正位是维持平衡的智慧，逆位是过度控制（对 Lily 的过度保护）——两者都是"节制"
- Alice（月亮）的正位是直觉和探索欲，逆位是迷失在幻觉中无法分辨真假——两者都是"月亮"

### 3.4 两个固定锚点

**XII 倒吊人 = 爷爷**
- 自愿倒悬：牺牲自我获得守护力量
- 新视角：成为God后看到了村子的全貌，但这个"全知视角"本身也是一种诅咒
- 倒吊人的微笑：爷爷在结局2灵魂消散时是微笑的——倒吊人牌上的人总是在微笑

**XV 恶魔 = 恶魔**
- 锁链是松的：契约是自愿签订的，随时可以终止（代价是村子毁灭）
- 恶魔不主动诱惑——是人主动来找他做交易
- 正位恶魔：忠实履行契约，你要守护我给你守护
- 逆位恶魔：交易的代价在不知不觉中吞噬一切

## 4. 时间系统

### 4.1 设计理念

参考星露谷效应：玩家需要足够的时间建立情感连接，才能让后续的揭示产生冲击。5天时间不够爱上一个村子，但一个完整的季节可以。

时间跨度以**季节**为单位，每季28个游戏日，全年4季。一个完整的叙事弧跨越约4个季节（一个游戏年），但结局可以在第4季之后的任何时间触发。

### 4.2 季节与叙事阶段

#### 第一季：春——播种（secret_stage=0）

**氛围**：温馨、充满希望、新的开始

- **日1-7**：抵达与安顿。认识所有NPC，熟悉村子地理，开始种田/采矿的日常
- **日8-14**：融入社区。参与第一次集体活动（播种祭），体验Carol的仪式（此时只是"有趣的传统"）
- **日15-21**：建立初步信任。至少和2-3个NPC有了较深的对话
- **日22-28**：第一季尾声。Lily第一次说出一句奇怪的话（"为什么没有人来我们村子玩？"），但很容易被忽略

**关键事件**：
- 日2：Lily首次出场——"你就是回来的人吗？Carol阿姨说你爷爷以前住在这里"
- 日5：Marco热情搭话——"你也是旅行者？我也是，不过我明天就走了"
- 日10：第一次播种祭，Carol主持，Dave在旁边念叨奇怪的祝词
- 日14：Bob沉默地给玩家一块好矿石——不说话，放下就走
- 日21：Alice在闲聊中提到"手册里有张旧纸，不知道是什么"

**God旁白风格**："雨水滋润了土地，万物在阳光下生长。""今天的风很温柔。"（平淡自然观察，像一个慈祥的旁白者）

#### 第二季：夏——生长与异变（secret_stage=1）

**氛围**：日常中出现微妙的不安，像夏天午后远处的闷雷

- **日29-35**：微小异常。西边废弃区域的空气有时会颤抖，偶尔有NPC提到奇怪的梦
- **日36-42**：异常升级。老井夜里传来低语声；夜间出现短暂诡异光源
- **日43-49**：个人异常。陈婆编织时突然停下看着主人公；Erik说修缮的老物件"好像有心跳"
- **日50-56**：夏末不安。一次小地震；Lily画了一幅"天上有眼睛在看我们"的画

**关键事件**：
- 日30：西边废弃农庄方向传来奇怪的声响（仅夜间）
- 日35：NPC们开始偶尔提到噩梦——内容各不相同但都和"被关在什么里面"有关
- 日40：老井夜里传来低语——走近听像是叹息
- 日42：爷爷小屋里发现一张老照片，上面有个看不清脸的人站在村子边缘
- 日47：陈婆看到主人公时突然停下编织——"你的脸……好像让我想起什么事"
- 日52：Erik修缮Dave带来的老物件时说——"这些东西拿在手里好像有心跳"
- 日55：第一次"红色黄昏"——天空短暂变红（天灾前兆的最初迹象）

**God旁白风格**："风中有不安的气息……但不要怕。还不需要怕。""注意那口井。不，别在意。我说什么了？"（流露紧张和保护欲，偶尔出现"失言"）

#### 第三季：秋——收获与碎裂（secret_stage=2）

**氛围**：信任建立后真相碎片浮现，像秋天的落叶——美丽但预示着衰亡

- **日57-63**：NPC开始主动透露碎片。前提是信任度已建立（玩家如果第一季就开始社交，此时正好）
- **日64-70**：线索链开始形成。玩家可以开始串联不同NPC的碎片信息
- **日71-77**：关键揭示。Dave讲出寓言故事；Alice拿出地图碎片；Bob终于说出爷爷消失的目击
- **日78-84**：秋末危机。天灾前兆加剧——红雪首次降临，持续一整天

**关键事件**（需要信任度达标才会触发）：
- 信任30+：NPC偶然提起爷爷——说了一句奇怪的话后自己也愣住
- 帮助过3+NPC：Dave主动讲古——"很久以前有人为了保护他爱的人把自己变成了一面墙……"
- 探索过废弃农庄：Bob警告——"你不应该去那里。我父亲……他也去了那里"
- 信任60+：Alice拿出地图碎片——草药手册里的地图指向法阵核心
- 信任50+：Carol提到仪式——"每年这个时候我都要准备那些东西……为什么？我也不知道"
- 与Marco深入对话：发现他说的"去年"的事其实是十年前的
- 帮Erik寻找材料：Erik发现箱子里的笔记——"地下室那个箱子里有手写图纸，和Dave让我修的东西一模一样"
- 拜访陈婆3次+：Lily跑来说"陈婆婆做梦喊了你爷爷的名字"
- 日70+：Lily画了一幅画——画里有被光包围的老人站在村子上方

**God旁白风格**："你越来越接近了。我不知道该为此高兴还是害怕。""那张地图……你看到了。好。不，不好。我不知道。"（矛盾情感浮现，守护者的内心挣扎外溢）

#### 第四季：冬——真相与抉择（secret_stage=3）

**氛围**：寒冷、沉重、不可回避。天灾迫在眉睫，一切隐藏的东西都暴露在冬日的白雪下

- **日85-91**：天灾前兆全面爆发。红雪反复降临，地面震动频繁，老井的低语变成清晰的词句
- **日92-98**：Dave揭示全部真相。条件是信任度>80且关键线索收集足够
- **日99-105**：准备期。根据已知信息，玩家准备应对天灾（此时三个结局路径分歧）
- **日106-112**：最终抉择。天灾到来，玩家在法阵核心做出最终选择

**关键事件**：
- 信任70+且日85+：天灾前兆——"红雪"大规模降临，村民开始恐慌
- 信任80+：Dave揭示全部真相——爷爷的身份、恶魔契约、结界本质
- 探索村子边界：发现看不见的墙壁，边缘有裂纹般的光
- 信任70+且Erik已看笔记：Erik理解——"这些不是装饰……是某种阵法的组件"
- 信任80+：陈婆记忆恢复——"我想起来了……那天晚上你爷爷走进了一团黑暗里就再也没出来"
- 所有核心线索收集完毕：最终抉择触发，三个结局分支点出现

**God旁白风格**："孩子……无论你做什么选择，都要知道——一切都是因为我爱你。""我已经说了太多了。但时间不多了。"（坦诚面对，守护者放下伪装）

### 4.3 时间节奏的灵活性

以上时间线是**建议节奏**，不是硬性脚本。实际进度取决于：

- 玩家的探索意愿（不探索的玩家可能到第二年才进入真相期）
- 信任度积累速度（完全不社交则永远停在stage 0）
- 天灾前兆是由God Agent根据契约衰退速率动态触发的，有内在时间压力

**关键设计**：即使玩家完全不推动叙事，天灾也会在约**第6季（日168左右）**自动到达临界点，届时God会通过天气和事件强制推动叙事进入stage 3。这确保了游戏最终会走向结局。

### 5.1 — 0 愚者：主人公

| 维度 | 内容 |
|------|------|
| **名字** | 由玩家决定 |

| **外貌意象** | 带着旧行囊，衣着简单但不属于村子的风格。脚边总有一条看不见的裂缝（结界的隐喻） |
| **公开性格（正位）** | 好奇、开放、勇敢。愿意尝试一切，对所有人友善，不预设立场 |
| **阴影面（逆位）** | 鲁莽、不听警告、可能因好奇伤害到别人。愚者的无知既是勇气也是危险 |
| **隐藏愿望** | 寻找归属感。离开城市是因为找不到归属，但"回来"这件事本身也是爷爷安排的 |
| **与秘密的关联** | 主角是秘密的核心——爷爷安排他回来就是为了应对天灾。他的存在本身就是契约的变量 |
| **话题禁区** | 无——愚者没有禁区，这是他能发现真相的根本原因 |
| **认知屏障强度** | 弱。刚回到村子，结界对他的认知影响尚未完全生效（正在渐染中） |
| **日常行程** | 白天种田/采矿/社交；黄昏整理；夜晚探索或休息 |
| **与爷爷的关系** | 爷爷的孙子。童年在这里度过，记忆温馨但模糊。对爷爷的"消失"有疑问但从未深究 |
| **三结局状态** | 正常：留下来，不再想离开（屏障完全生效）；隐藏1：代替爷爷成为新的守护者；隐藏2：获得自由但失去爷爷 |
| **对应道具** | 旧行囊（愚者之囊）——见道具章节 |

### 5.2 — II 女祭司：陈婆

| 维度 | 内容 |
|------|------|
| **名字** | 陈婆（保留。"陈"谐音"沉"——沉入遗忘的记忆） |
| **牌面** | II The High Priestess — 坐在两根柱子之间，一手持卷轴，帷幕遮住身后的水 |
| **职业** | 村子里最年长的女性，编织者。不务农不经商，只编织 |
| **外貌意象** | 银发梳成整齐的髻，手指因常年编织而粗糙但灵活。眼神偶尔突然清亮，像帷幕被风吹开一瞬 |
| **公开性格（正位）** | 慈祥、温和、总是在编织。对年轻人有一种天然的亲和力，喜欢叫路过的人坐下聊天 |


## 7. 11个道具完整设计

每张牌对应一个特殊道具。共11件，每件有正用（利他/探索）与逆用（利己/回避），且使用有trade-off。

---

### 7.1 — 愚者之囊（The Fool's Satchel）

| 维度 | 内容 |
|------|------|
| **对应** | 0 愚者 / 主人公 |
| **外观** | 一个旧帆布行囊，角落绣着褪色的花纹。打开后比外面看起来大得多（爷爷曾用过，里面残留着微弱的结界力量） |
| **来源** | 主人公回村时带来的，是爷爷寄给他的"遗物" |
| **正用** | 触碰NPC时唤醒一段与爷爷相关的模糊记忆碎片（加速线索揭示） |
| **逆用** | 把东西放进囊中，那件东西会被"遗忘"——其他NPC不再记得它的存在（可以藏匿线索避免被发现） |
| **Trade-off** | 正用每次使用会让主角的认知屏障加深一点（越接近真相，结界反噬越强）；逆用每次使用会让一个NPC对主角的信任度降低（他们感觉主角在隐瞒什么） |
| **叙事联动** | 隐藏结局2需要把"愚者之囊"放在法阵核心——它是爷爷和主角的联系物，是破除契约的情感锚点 |

---

### 7.2 — 无尽之线（The High Priestess's Thread）

| 维度 | 内容 |
|------|------|
| **对应** | II 女祭司 / 陈婆 |
| **外观** | 一团永远织不完的丝线，颜色随光线变化。线的图案和法阵完全一致 |
| **来源** | 陈婆手中，她不记得从哪里来的。"手停不下来" |
| **正用** | 将线铺在法阵组件上，可以短暂看到完整的法阵图案（揭示结界结构） |
| **逆用** | 将线缠绕在某人身上，可以暂时封锁他们的一段记忆（制造新的遗忘） |
| **Trade-off** | 正用会让陈婆头疼加剧（她的潜意识在被撕开）；逆用会让线变短一截——用完了，陈婆最后的记忆连接也会断裂 |
| **叙事联动** | 正常结局中陈婆用线修补法阵裂缝；隐藏结局2中线自行解开——代表记忆封印的解除 |

### 7.3 — 父亲的矿镐（Strength's Hammer）

| 维度 | 内容 |
|------|------|
| **对应** | VIII 力量 / Bob |
| **外观** | 一把沉重的旧矿镐，手柄上有Bob父亲刻的名字。镐头有种不自然的寒意 |
| **来源** | 在废弃矿山深处找到——Bob父亲失踪时留下的 |
| **正用** | 可以采掘普通工具无法开采的"结界矿石"——法阵修复/重铸的关键材料 |
| **逆用** | 砸向法阵组件可以造成裂缝——加速结界衰退（但这也意味着加速天灾到来） |
| **Trade-off** | 正用每次会触发Bob关于父亲的闪回记忆（情绪伤害但也是治愈的过程）；逆用每次使用会让矿镐上的名字逐渐消失——当名字完全消失，Bob会永远失去关于父亲的记忆 |
| **叙事联动** | Bob接受父亲的真相后，矿镐上的名字会重新发光——代表力量从恐惧回归勇气 |

### 7.4 — 守夜人之灯（The Hermit's Lantern）

| 维度 | 内容 |
|------|------|
| **对应** | IX 隐者 / Dave |
| **外观** | 一盏造型古朴的铜灯笼，火焰是冷蓝色的。不需要燃料，自行燃烧 |
| **来源** | Dave一直拎着的灯笼。爷爷留给他的，是守护者力量的微小碎片 |
| **正用** | 在夜间照亮隐藏的线索——法阵组件发光、结界裂缝显形、NPC梦话显现为可读文字 |
| **逆用** | 熄灭灯笼可以让一个区域"隐藏"——那个区域对所有NPC不可见，包括God的感知 |

### 7.5 — 倒悬的怀表（The Hanged Man's Watch）

| 维度 | 内容 |
|------|------|
| **对应** | XII 倒吊人 / 爷爷 |
| **外观** | 一只银色怀表，指针反向转动。表面刻着倒悬的树。打开后内侧有一张极小的全家福——年轻的爷爷、奶奶和作为婴儿的主角 |
| **来源** | 在爷爷的小屋隐藏夹层中找到。这是爷爷签订契约时随身携带的唯一私人物品 |
| **正用** | 让时间在一小片区域短暂"倒流"——可以看到过去发生的事件残影（关键回忆的可视化） |
| **逆用** | 让时间在一小片区域"加速"——加速植物生长/矿石再生，但也加速NPC的衰老和记忆衰退 |
| **Trade-off** | 正用每次让怀表的指针慢一点（使用次数有限，指针停止时怀表永远沉默）；逆用每次让怀表的全家福更模糊一点（直到完全看不清） |
| **叙事联动** | 三个结局中怀表的状态不同：正常结局——指针继续反转；隐藏1——指针停止，全家福清晰；隐藏2——怀表碎裂，碎片化为光点飘散 |

### 7.6 — 不灭之灶（Temperance's Hearth）

| 维度 | 内容 |
|------|------|
| **对应** | XIV 节制 / Carol |
| **外观** | Carol厨房里的老灶台，火焰从未熄灭过。仔细看，火焰的形状像两个杯子之间倾倒的水 |
| **来源** | 村子建立时就在的灶台。Carol继承了维护它的责任。灶火实际上是法阵的一部分——热量维持着结界的温度 |
| **正用** | 在灶火上烹制的食物有治愈效果——不只是恢复体力，还能暂时抚平NPC的心理创伤和认知混乱 |
| **逆用** | 在灶火上焚烧某物，可以永久消除一段"公共记忆"——所有NPC同时遗忘这件事 |
| **Trade-off** | 正用消耗灶火能量（火焰变小=结界变薄=天灾更容易渗透）；逆用的代价是Carol会出现一段空白时间（她会发呆几个小时，这段时间Lily没人管） |
| **叙事联动** | Carol-Lily支线高潮：Carol发现灶火的真正意义后的选择——维持灶火=维持笼子，熄灭灶火=解放但失去"家"的温暖 |

### 7.7 — 锁链之契（The Devil's Chain）

| 维度 | 内容 |
|------|------|
| **对应** | XV 恶魔 |
| **外观** | 一段暗色金属链条，表面刻有无法辨认的文字。拿在手里沉重得不合理。链条两端各有一个开口的环——可以扣上，也可以取下。**链子是松的** |
| **来源** | 法阵核心深处，契约的物理化身。只有在stage 3、法阵核心被揭示后才能接触到 |
| **正用** | 将链条扣在法阵核心上，可以临时加固结界——抵御天灾的应急手段 |
| **逆用** | 将链条戴在自己身上，可以短暂获得恶魔般的力量——超自然的感知和行动能力 |
| **Trade-off** | 正用加固结界=加固禁锢，玩家使用后会感到一阵深深的疲惫和空虚（主角在复制爷爷的选择）；逆用力量有代价——每次使用后一段时间内无法感受到任何情感（预演隐藏结局1的代价） |
| **叙事联动** | 正常结局用它加固法阵；隐藏结局1用它与恶魔沟通谈判新条件；隐藏结局2用恶魔真名使链条碎裂——契约物理载体被摧毁 |

### 7.8 — 不指北的罗盘（The Tower's Compass）

| 维度 | 内容 |
|------|------|
| **对应** | XVI 塔 / Marco |
| **外观** | 一只精致的铜罗盘，指针永远旋转不停——在结界内，它找不到"北"。外壳上刻着一座塔的浮雕 |
| **来源** | Marco随身携带的"旅行必备"。他每天都看，但从未注意到指针一直在转 |
| **正用** | 带到村子边缘，罗盘会指向结界最薄弱的点（帮助发现边界裂缝） |
| **逆用** | 对着NPC使用，可以让他们暂时意识到自己被困——但这种突然的觉醒会造成极大的心理冲击 |
| **Trade-off** | 正用每次让Marco的时间感知更加混乱（他可能开始说"我昨天才来的"——实际上已经二十年）；逆用造成的心理冲击如果不妥善处理，NPC可能陷入恐慌甚至精神崩溃 |
| **叙事联动** | 隐藏结局2中结界消失，罗盘第一次指向真正的北方。Marco手持罗盘站在村口——终于可以出发了，但世界已不是他记忆中的样子 |

---

### 7.9 — 星光蜡笔（The Star's Crayon）

| 维度 | 内容 |
|------|------|
| **对应** | XVII 星星 / Lily |
| **外观** | 一支银色蜡笔，只在Lily手里才会发光。画出的线条在黑暗中像星光一样闪烁 |
| **来源** | Lily在老井旁边捡到的，她觉得"这是那个老爷爷给我的"（实际上是守护者力量的微小碎片） |
| **正用** | Lily画出的画有预言性质——她画的东西会隐喻接下来要发生的事件（玩家可以从画中获得预知线索） |
| **逆用** | 玩家如果拿走蜡笔自己使用，可以在任何表面画出临时的保护符文——但符文只在黑夜生效 |
| **Trade-off** | 正用依赖Lily的直觉（如果Lily情绪不好或被Carol限制外出，画的准确性下降）；逆用=从孩子手中夺走她的"星星"，Lily会变得安静和消沉，失去部分直觉能力 |
| **叙事联动** | Lily在正常结局中长大后忘记蜡笔的特殊性，把它放在了抽屉深处。隐藏结局2中Lily用蜡笔画了最后一幅画——"爷爷在笑" |

---

### 7.10 — 半张地图（The Moon's Page）

| 维度 | 内容 |
|------|------|
| **对应** | XVIII 月亮 / Alice |
| **外观** | 一张泛黄的旧纸，上面是手绘的地图，但只有半边——另一半被撕掉了。月光下看地图上的线条会移动 |
| **来源** | Alice的草药手册里夹着的。写手册的人（爷爷）故意留下的线索 |
| **正用** | 在月光下展开，地图会显示法阵核心的大致位置和路径（但因为只有半张，位置有偏差） |
| **逆用** | 将地图覆盖在某个区域上，可以让该区域"隐藏"——从地图上消失意味着从认知中消失 |
| **Trade-off** | 正用：月光下阅读会被月亮的幻觉影响——主角可能看到不存在的路径或虚假的目标（需要和其他线索交叉验证）；逆用：被隐藏的区域如果有NPC在其中，该NPC会进入迷失状态 |
| **叙事联动** | 另一半地图在Erik的地下室箱子里（爷爷的锻造笔记中）。两半合一才能精确定位法阵核心。这联动了Alice和Erik两条线索链 |

### 7.11 — 无名之砧（The Sun's Anvil）

| 维度 | 内容 |
|------|------|
| **对应** | XIX 太阳 / Erik |
| **外观** | Erik铁匠铺里的那块老铁砧，表面有无法磨去的奇怪花纹。只有在锻造法阵组件时，花纹才会发出微弱的金光 |
| **来源** | 和铁匠铺一起从上一任铁匠继承的。实际上是爷爷特意安排的——铁砧本身就是最大的法阵组件 |
| **正用** | 在铁砧上锻造的物品天然带有微弱的结界力量——法阵组件在这上面修缮效果加倍 |
| **逆用** | 砸碎铁砧可以释放储存的大量结界能量——一次性的巨大力量爆发 |
| **Trade-off** | 正用：每次锻造法阵组件Erik会感到一种"被抽干"的疲惫（他在无意识地注入自己的生命力）；逆用：铁砧一旦碎裂不可恢复，Erik也会永久失去锻造的能力 |
| **叙事联动** | 隐藏结局1中Erik用铁砧锻造新的契约载体——砧面上的花纹变成新的法阵图案。正常结局中铁砧继续沉默地运转。隐藏结局2中结界消失后铁砧裂开——Erik终于明白自己多年的"满足感"来自何处 |

## 8. 三个结局完整设计

---

### 8.1 正常结局：轮回（大多数玩家）

**触发条件**：
- secret_stage 到达 3
- 信任度 >= 50
- 收集了核心线索（法阵位置、基本材料配方）
- 最终抉择选择"加固法阵"

**过程**：
1. Alice 的半张地图 + Erik 笔记中的另一半合成完整地图，找到法阵核心
2. Bob 用父亲的矿镐采集结界矿石
3. Erik 在无名之砧上锻造新的法阵组件
4. Carol 按照Dave指导的仪式将材料放置在正确位置
5. Dave 主持最后的加固仪式——他第一次在众人面前用接近直白的方式说话
6. 陈婆的无尽之线铺在法阵裂缝上，编织图案与法阵完美贴合

**结尾画面**：
- 天灾被抵御，红雪停止，天空恢复正常
- 结界重新加固，比以前更坚固
- 村民们庆祝，但异常记忆在几天内逐渐模糊
- **最黑暗的细节**：主人公发现自己已经不想出去了。站在村口看着外面的路，觉得"那边有什么？算了，这里挺好的"——认知屏障终于完全生效
- God最后旁白："你留下来了……谢谢你。但原谅我，我不知道该为此高兴还是悲伤。"
- 画面渐暗，最后一个镜头：Marco在村口说"明天就走了"，Lily蹦蹦跳跳跑过——但她今天没有抬头看天空

**哲学含义**：守护的延续也是禁锢的延续。爱赢了，自由输了。最可怕的是你甚至不知道自己输了。

---

### 8.2 隐藏结局1：新交易（二周目解锁）

**触发条件**：
- 已完成正常结局一次
- secret_stage = 3
- 信任度 >= 90
- 收集全部线索（含核心+隐藏）
- 找到Erik地下室箱子并阅读锻造笔记
- 最终抉择选择"寻找另一条路"

**过程**：
1. 主人公利用锻造笔记和Erik的帮助，在无名之砧上制作新的契约载体
2. 带着锁链之契到法阵核心，通过通道联系恶魔
3. **面对面谈判**——恶魔显现，两条松松的链子垂在身侧："又一个来做交易的人？你爷爷也是这样坐在这里的。条件还是一样：你想要什么，你愿意付什么？"
4. 主人公提出新条件：用自己的代价换取更强的力量，不依赖爷爷的旧契约
5. 恶魔同意——"有意思的交易。你比你爷爷年轻，所以代价……也会不一样"

**代价**：主人公**永远无法感受到"爱"的情感**。可以理性知道什么是爱，可以判断谁在关心自己，但再也无法真正*感受*到温暖、思念、依恋。

**结尾画面**：
- 天灾被新力量抵御
- 旧契约解除，爷爷灵魂获得自由——但他选择以普通灵魂形式留在村子（不再是守护者，只是一个安静的存在）
- 结界仍在——新契约同样需要结界，但认知控制减弱了
- 所有人都能模糊地感觉到"有什么不一样了"
- **最后一幕**：黄昏，主人公站在村口看夕阳。Lily跑过来拉他的手——"回去吃饭啦！"他低头看着Lily的笑脸，知道这是美好的画面，但心里只有平静的空白
- God最后旁白："你打破了我的锁，但锻了一把新的钥匙。孩子……你会比我更好地守护他们的。"——声音里有欣慰，也有心疼

**哲学含义**：打破了爷爷的笼子，建了自己的笼子。代价从一个人转移到另一个人。轮回还是解脱？答案开放。但有一点是确定的：这次是主角**自己的选择**。

---

### 8.3 隐藏结局2：终结（三周目解锁，最沉重的选择）

**触发条件**：
- 已完成隐藏结局1一次
- secret_stage = 3
- 信任度 = 100（所有NPC最大信任）
- 收集全部线索 + 完整理解恶魔契约运作方式
- 通过多周目积累线索拼出恶魔的真名
- 最终抉择选择"终结一切"

**恶魔真名获取**：
- 真名碎片分散在6个地方：Dave的故事（2片）、陈婆的编织图案（1片）、锻造笔记（1片）、老井回声（1片）、怀表内刻字（1片）
- 每个碎片在不同周目才能被发现（第一周目能找到2个，第二周目2个，第三周目2个）
- 6片拼成完整的真名

**过程**：
1. 在法阵核心直面恶魔，说出真名
2. 恶魔显形——不是愤怒，是意外："有意思。很久没有人记得我的名字了。你知道说出这个名字意味着什么？"
3. 主角说："我知道。"
4. 恶魔："你爷爷的灵魂绑在契约上。契约消失，他也消失。你确定？"
5. **主角必须在知道这一点的情况下做出选择**
6. 真名的力量破除恶魔的存在——恶魔消散，契约消失

**结尾画面**：
- 恶魔消散，天灾根源清除
- 结界从村子边缘开始像玻璃一样碎裂——碎片化为光点飘向天空
- **爷爷灵魂消散前的最后一刻**：所有村民同时看到了他——一个微笑着的老人虚影站在村子上方。倒悬的姿态缓缓变为直立，脸上的微笑没有变
- 陈婆记忆完全恢复，跌坐在地上哭着喊出爷爷的名字——"那天晚上他走进了黑暗里，对我说别怕，他会保护我们所有人"
- Dave终于放下提了一辈子的灯笼，坐在地上——"他终于自由了。我们都自由了"
- Bob沉默地流泪，Erik第一次不知道该说什么
- Carol紧紧抱着Lily，第一次意识到她一直在重复一种模式
- Marco手持罗盘站在村口——指针第一次安静地指向了北方。他久久不动
- **Lily是唯一笑着说"爷爷再见"的人**——因为她一直都能看到他。她挥着手，眼泪和笑容一起落下
- 主人公站在打开的村口。面前是广阔的世界，身后是自由的村子，但再也没有爷爷了
- **没有God的旁白——因为他已经不在了。只有风声。**
- 最后的最后：风中传来一个极其微弱的声音——不是旁白，是残留的回声——"去吧。"

**哲学含义**：真正的自由总有代价，而那个代价往往是你最爱的人。但这是唯一一个**所有人都获得了选择权**的结局——包括爷爷自己。倒吊人终于被放下来了。

---

## 9. 支线设计

---

### 9.1 Carol-Lily 支线（主线精确缩影）

**归属角色**：Carol（XIV 节制）、Lily（XVII 星星）

**起点**（第一季）：
- 日常中观察到Carol对Lily的严格管教——不许去矿区、不许问"奇怪的问题"、不许和Marco太亲近
- Lily偶尔向主角抱怨："Carol阿姨不让我去看Erik叔叔打铁了"
- 表面是正常的养母管教

**转折**（第二季-第三季）：
- Lily开始说出越来越接近真相的话，Carol越来越焦虑
- Carol开始限制Lily和主角的接触——"别总是跟Lily聊那些有的没的"
- 关键场景：Carol发现Lily画的"天上的老爷爷"，撕碎了画纸。Lily哭了
- 如果主角介入：Carol爆发——"你不明白！她的父母就是因为问了太多问题才……"（突然沉默）
- 这是Carol第一次触碰到自己的认知屏障边缘

**结尾**（第四季，依据结局分支）：
- 正常结局：Carol的焦虑消退，Lily变得更安静——"长大了懂事了"（实际是直觉被磨灭）
- 隐藏1：Carol和Lily达成新的平衡——Carol学会放手一点，Lily学会理解保护的善意
- 隐藏2：Carol和Lily一起站在打开的村口。Carol终于松开紧握Lily的手——"你想去哪里都可以"

**与主线耦合**："过度保护=禁锢"在Carol-Lily的小尺度上完整演绎了爷爷-村子的大叙事。玩家在这条支线中经历的情感，会直接影响他对主线的理解。

---

### 9.2 Bob 的父亲支线

**归属角色**：Bob（VIII 力量）、Erik（XIX 太阳）

**起点**（第一季）：
- Bob偶尔提到父亲，但立刻转移话题
- 在废弃矿山方向，Bob的步伐会变慢——他想去又不敢

**转折**（第二季-第三季）：
- 帮Bob探索废弃矿山，发现父亲的矿镐和一本工作日志
- 日志最后几页记录："矿山深处有东西……不是矿石。是声音。是光。我必须弄清楚"
- Bob的父亲发现了法阵的一部分，试图调查，然后失踪了——他穿过了一个结界裂缝，去了"外面"
- 但结界封闭了裂缝，他再也回不来了

**结尾**：
- Bob终于知道父亲不是死了，而是被困在"外面"——和村子里的人恰好相反
- 隐藏结局2中结界消失，有可能（暗示但不确认）Bob的父亲还在某处

**与主线耦合**：Bob父亲的命运是结界双面性的直接证据——结界既困住了里面的人，也阻止了外面的人回来。

---

### 9.3 Dave-陈婆 记忆支线

**归属角色**：Dave（IX 隐者）、陈婆（II 女祭司）

**起点**（第一季）：
- 两个老人经常坐在一起，但对话总是充满了停顿和未完成的句子
- "那天晚上……""什么？""什么什么？我说什么了？"

**转折**（第二季-第三季）：
- 随着结界衰弱，陈婆偶尔在梦中喊出爷爷的名字
- Dave知道这意味着封印在松动，他既希望陈婆想起来（这样她就不再痛苦），又害怕想起来（记忆本身太沉重）
- 关键场景：某个黄昏，陈婆突然停下编织，看着Dave说了一句清晰的话——"他让我别忘了，但我还是忘了"。Dave的眼泪无声地落下

**结尾**：
- 正常结局：陈婆的编织图案在法阵加固后变回了普通花纹。Dave在旁边看着，沉默
- 隐藏2：陈婆记忆完全恢复。她从头到尾说出那天晚上的事——这是整个游戏中最长的独白，也是最心碎的

**与主线耦合**：知道全部真相的人和忘记全部真相的人——这对关系是对"认知屏障"机制最诗意的诠释。

---

### 9.4 Erik 的"完美作品"支线

**归属角色**：Erik（XIX 太阳）

**起点**（第一季）：
- Erik对自己的作品总是不满意——"差了点什么"
- 他给主角展示作品时充满自豪，但夜里独自看着铁砧发呆

**转折**（第二季-第三季）：
- 发现锻造笔记后，Erik按照笔记中的方法尝试锻造——完成后第一次说"……就是这种感觉"
- 他意识到"完美"的标准来自于铁砧本身——铁砧在"教"他锻造法阵组件
- 关键场景：Erik第一次主动打开地下室的箱子

**结尾**：
- 隐藏1：Erik为主角锻造新的契约载体——这是他一生中最满意的作品
- 隐藏2：铁砧裂开后Erik站在碎片前——"原来这些年……是你在教我"

**与主线耦合**：Erik无意识地维护着结界的物理结构，他对"完美"的追求实际上是被结界力量驱动的。当他理解真相时，必须重新定义自己的手艺意味着什么。

---

### 9.5 Marco 的时间支线

**归属角色**：Marco（XVI 塔）

**起点**（第一季-第二季）：
- Marco讲的故事细节和时间线不匹配
- Lily天真地指出——"Marco叔叔，你说明天走，但你昨天也这么说了呀"

**转折**（第三季）：
- 玩家有意识地对比Marco说的"最近"事件和实际时间线
- 发现他口中"去年路过的城市"发生的事件是十年前的新闻
- 关键场景：主角直接问Marco——"你到底来了多久了？"Marco的表情从困惑变成恐惧——塔在崩塌

**结尾**：
- Marco意识到自己被困的真相是整个游戏中最戏剧化的"塔"时刻
- 隐藏2中他面对重获的自由：外面的世界已经不是他记忆中的样子——他的家人可能已经老去甚至不在了

**与主线耦合**：Marco是结界存在的活体证据。他的遭遇从"滑稽的口头禅"变成"最残忍的真相"——时间被偷走了，而他一无所知。

---

### 9.6 Alice 的月下之路支线

**归属角色**：Alice（XVIII 月亮）

**起点**（第一季）：
- Alice偶尔在夜晚看手册里的地图出神
- 她说不清为什么那张纸让她心神不宁

**转折**（第二季-第三季）：
- 月光下展开地图，上面的线条似乎在移动
- Alice开始在夜晚循着地图探索——但月亮的幻觉让她看到不存在的路径
- 关键场景：Alice深夜独自来到废弃农庄——看到了"什么"（法阵组件在月光下发出微光），但第二天记忆模糊了

**结尾**：
- 当两半地图合一（Alice的半张+Erik箱子里的半张），Alice终于能在月光下看清完整的路
- 隐藏2中Alice带着完整地图站在村口——"原来外面一直就在这里"

**与主线耦合**：月亮之路=在幻觉与真相之间的探索。Alice的支线是玩家收集线索的主要路径之一，同时也是对"看得见但看不清"这个主题的具象化。

## 10. 核心机制叙事集成

### 10.1 认知屏障渐染

**机制**：玩家的认知屏障会**随时间渐染加深**。

**触发条件**：连续多天不探索异常、不与NPC深入对话、不推动叙事。

**表现层级**：
1. **对话选项减少**：探索性/质疑性的选项逐渐消失，只剩下日常选项
2. **视野边界变暗**：屏幕边缘出现越来越浓的暗色晕影
3. **God旁白变安抚**："今天天气真好，不需要想那么多。""安心待在这里就好。"
4. **NPC异常减少**：NPC不再说出奇怪的话，一切变得"正常"
5. **极端情况**：如果玩家完全不探索持续两个季度，会出现"你觉得这个村子就是全世界"的认知提示——这是正常结局的前兆

**叙事含义**：这就是结界的运作方式——不是暴力封锁，而是温柔地让你忘记外面的存在。玩家亲身体验认知屏障，而不只是听NPC描述。

**可逆性**：任何时候开始探索/深入对话都能逐步恢复——但恢复速度比渐染速度慢。

### 10.2 God 的沉默即旁白

**机制**：God Agent 在特定场景中选择**不说话**，沉默本身就是旁白。

**场景映射**：

| 场景 | 沉默含义 | 情感 |
|------|---------|------|
| 玩家靠近Marco | 爷爷的愧疚——Marco被困是他造成的 | 无法面对自己的过错 |
| 玩家伤害NPC | 失望——但不是对"错误"的失望，是"我保护的人在互相伤害" | 深沉的悲伤 |
| 玩家连续不探索 | 安抚——他其实*希望*你不要发现真相，这样更安全 | 矛盾的保护欲 |
| 玩家到达法阵核心 | 恐惧——你走到了最接近真相的地方 | 害怕失去 |
| 天灾前兆发生时 | 全力维持结界，无暇旁白 | 拼尽全力 |
| 隐藏结局2的最后 | 永久沉默——他不在了 | 终结 |

### 10.3 白天/黑夜双面玩法

**白天：温馨田园**
- 种田、采矿、社交、交易——标准沙盒模拟
- NPC展现正位特质，世界明亮安全
- 适合建立关系、收集资源、推进日常

**黄昏：过渡带**
- NPC行为开始微妙变化——Bob望向西方、Carol检查仪式材料、Dave开始巡视
- 世界色调变暖变沉
- 某些对话只在黄昏触发（NPC更放松/更脆弱的时段）

**黑夜：克苏鲁层**
- 天灾前兆活跃——异常光源、低语、红雪
- NPC展现逆位特质或做噩梦
- 法阵组件在月光下可见
- 废弃区域出现异常实体（天灾渗透的具象化）
- **战斗发生在这里**：玩家需要对抗天灾渗透带来的实体威胁

**叙事集成**：白天越美好，夜晚的反差越大。玩家最终意识到——白天的"美好"本身就是结界维护的假象。

### 10.4 NPC行为频率分级

| 级别 | 角色 | 视野内频率 | 视野外频率 | 说明 |
|------|------|-----------|-----------|------|
| **主要** | Alice, Bob, Carol, Dave, Erik, Lily | 每5-10秒思考 | 每30秒思考 | 完整LLM控制 |
| **次要** | Marco, 陈婆 | 每15-30秒思考 | 每90秒思考 | 简化LLM，think_interval_multiplier: 3.0 |
| **God** | 守护者 | 每tick监控 | 每tick监控 | 叙事导演，不受视野影响 |
| **恶魔** | 无名者 | 不适用 | 不适用 | 仅在特定叙事节点脚本触发 |

### 10.5 视野减频与探索激励

**机制**：不在玩家视野内的NPC行为频率降低，但不消失。

**叙事集成**：
- 你看不见的时候NPC也在生活——回来后他们会提到你不在时发生的事
- 如果长时间不见某个NPC，再见面时他们的变化会更大（积累的状态变化一次性呈现）
- 特定事件只在玩家不在场时发生——例如Dave和陈婆的黄昏对话，玩家需要特意不在场然后从Lily那里听到转述
