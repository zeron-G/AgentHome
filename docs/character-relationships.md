## 认知屏障等级

| 角色 | 屏障强度 | 表现 |
|------|---------|------|
| **Marco** | 最强 | 连时间感知都被扭曲 |
| **Alice, Bob, Carol, Erik** | 强 | 标准屏障，从未想过离开 |
| **陈婆** | 中 | 记忆被刻意模糊但有潜意识残留 |
| **Dave** | 弱 | "古怪"使他更接近真相 |
| **主人公** | 弱（渐染中） | 刚回来，屏障尚未完全生效 |
| **Lily** | 最弱/无 | 孩子对屏障有天然抵抗力 |

## 三结局角色状态总览

| 角色 | 正常结局 | 隐藏结局1 | 隐藏结局2 |
|------|---------|----------|----------|
| **主人公** | 留下，屏障完全生效 | 成为新守护者，失去爱的感受 | 获得自由，失去爷爷 |
| **爷爷** | 继续守护，高兴又悲伤 | 灵魂自由，选择留下 | 灵魂消散，微笑着 |
| **Dave** | 继续守夜，"又一个轮回" | 担忧但尊重 | 终于放下灯笼 |
| **陈婆** | 继续编织遗忘 | 记忆部分恢复 | 记忆完全恢复，哭述一切 |
| **Bob** | 恐惧渐淡，安于现状 | 主动帮忙挖掘材料 | 知道父亲真相，释然 |
| **Erik** | 继续修缮，觉得有意义 | 锻造"完美作品" | 铁砧裂开，重新定义手艺 |
| **Carol** | 继续仪式，莫名安心 | 震惊但选择理解 | 自由但空虚，知道Lily父母真相 |
| **Lily** | 长大后忘记直觉 | 兴奋又害怕 | **唯一笑着说"爷爷再见"的人** |
| **Alice** | 安于村中，渴望消失 | 提供草药帮助 | 终于可以去外面学医 |
| **Marco** | 继续说"明天走" | 发现真相，选择留下帮忙 | 罗盘指北，但世界已变 |
| **恶魔** | 不出现 | 达成新交易 | 被真名消灭 |
