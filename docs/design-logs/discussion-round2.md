# AgentHome 塔罗重构 — 完整设计讨论

## 一、11张牌的选定与论证

### 选牌的底层逻辑

不是"什么牌好看"，而是：**哪11张牌之间的象征关系能自然构成这个故事的叙事网络？** 塔罗大阿卡那本身就是一个旅程（愚者之旅），我们从中截取最能映射"守护/禁锢、交易/代价、自由/选择、记忆/遗忘"的11个节点。

### 最终选定的11张

| 序号 | 牌 | 角色 | 选择理由 |
|------|-------|------|----------|
| **0** | **愚者 The Fool** | 主角（玩家） | 固定。旅程的起点，天真的闯入者 |
| **II** | **女祭司 The High Priestess** | 陈婆（记忆守门人） | 隐秘知识的持有者，帷幕两侧的守护者，持有她读不出的卷轴 |
| **VIII** | **力量 Strength** | 矿工（原Bob） | 不是蛮力，是驯服内心恐惧的勇气。牌面是女人温柔地合上狮口 |
| **IX** | **隐者 The Hermit** | 守夜人（原Dave） | 举灯引路的孤独者，持有真相却只能在黑暗中照亮一小步 |
| **XII** | **倒吊人 The Hanged Man** | 爷爷 / God | 自愿的倒悬，牺牲换取新视角。**最适合爷爷，下面详细论证** |
| **XIV** | **节制 Temperance** | 厨师/仪式守护者（原Carol） | 两个容器之间倾倒的液体——调和、传承、无意识的仪式维护 |
| **XV** | **恶魔 The Devil** | 恶魔（交易者） | 固定锚点。不是邪恶，是欲望的镜子和交易的中介 |
| **XVI** | **塔 The Tower** | 铁匠（原Erik） | 构建与摧毁，当发现自己多年维护的是囚笼时——塔的崩塌 |
| **XVII** | **星星 The Star** | 草药师（原Alice） | 创伤后的希望，一只手将水倒向大地（知识），一只手倒向水池（直觉） |
| **XVIII** | **月亮 The Moon** | 旅行者（原Marco） | 幻觉、错误的感知、被扭曲的现实。月亮下的路通向未知但路是歪的 |
| **XIX** | **太阳 The Sun** | 孩子（原Lily） | 赤裸的真相、无遮蔽的喜悦。太阳下没有阴影可以藏匿 |

### 关键选牌论证

**为什么爷爷是XII倒吊人而不是其他牌？**

倒吊人是大阿卡那中最矛盾的牌。他自愿倒悬于树上，一条腿弯曲形成十字——这不是受刑，而是**主动选择的牺牲**。他从倒置的视角看到了常人看不到的世界。

正位：自愿牺牲、放下执着、获得更高的视角、等待中的智慧
逆位：**不必要的殉道、自我感动的牺牲、用牺牲来控制他人**

这个正逆位落差正是爷爷角色的核心矛盾。他的牺牲是真实的爱——但逆位提出了残酷的问题：**他的牺牲是否也是一种自私？** 他选择了"保护所有人"而不是"让所有人自己选择面对危险"，这本身就是倒吊人逆位的典型表现——用殉道感来回避"放手"的恐惧。

对比其他候选：
- IV 皇帝（权威/控制）太直白了，缺少爷爷身上那种"无力中的挣扎"
- V 教皇（传统/教导）太被动，爷爷是主动做出了震撼的选择
- XX 审判（重生/觉醒）更适合结局，不是角色

**为什么恶魔就是XV恶魔？**

这不是偷懒——恰恰是因为XV恶魔牌本身就不是"邪恶"。

看牌面：两个人被锁链绑在恶魔的祭坛上，**但锁链是松的**。他们随时可以摘下锁链离开，但他们选择不离开。恶魔没有强迫任何人——他只是坐在那里，让你看到自己的欲望。

正位（荣泽的重定义下）：满足他人欲望的交易者、镜子般映照你真正想要的东西
逆位：利用欲望操纵、让人沉溺于短期满足而忽视长期代价

**恶魔的存在意义不是"作恶"，是"让你诚实面对自己到底想要什么"。** 爷爷在恶魔面前说"我想保护所有人"，恶魔说"好的，代价是他们的自由和你的灵魂"，爷爷说"成交"——这笔交易里，谁才是真正做了"恶"的那个？

### 哪些牌的正逆位落差最具叙事张力？

**第一梯队（最大张力）：**

1. **XII 倒吊人**（爷爷）：自愿牺牲 ↔ 殉道式控制。落差直指主题核心。
2. **XVI 塔**（铁匠）：信念结构的崩塌。正位是"必要的毁灭带来更新"，逆位是"恐惧改变，执着于废墟"。当Erik发现自己多年修缮的不是传统装饰品而是囚笼组件——这就是塔的闪电劈下来的瞬间。
3. **XVIII 月亮**（旅行者）：直觉与幻觉只有一线之隔。Marco活在完全的月光幻觉中——逆位的月亮意味着被困在自己的幻觉中无法醒来。

**第二梯队（强张力）：**

4. **VIII 力量**（矿工）：驯服恐惧 ↔ 被恐惧驯服。Bob每次想调查矿山就退缩，是逆位力量的完美体现。他的成长弧是从逆位翻回正位。
5. **II 女祭司**（陈婆）：直觉知识 ↔ 被封印的知识。女祭司坐在黑白两根柱子之间，持有TORA卷轴但用斗篷半遮——陈婆完全就是这个形象：她手里编织的图案就是"卷轴"，但她读不出来。

**第三梯队（稳定但可靠）：**

6. **XVII 星星**（草药师）：希望/治愈 ↔ 绝望/断联。Alice是最积极的角色，但星星逆位提醒我们：如果希望的源头（爷爷）消失了，她会如何面对？
7. **XIV 节制**（厨师）：和谐的调和 ↔ 过度/失衡。Carol的仪式维护是"无意识的调和"，但节制逆位暗示——当调和变成执念，仪式变成枷锁。

## 二、从牌出发的11个角色完整设计

我不再沿用旧名字。从每张牌的原型出发，重新设计。但保留核心叙事功能。

### 0 愚者 — 主角（玩家）

不需要预设性格。愚者的意义在于**他什么都不是，所以什么都可能成为**。愚者脚下是悬崖，但他不看脚下——这就是玩家回到村子时的状态：以为只是回乡，不知道脚下是深渊。

愚者的白色玫瑰代表纯真，小包袱代表"以为够用的准备"。这两样都会在游戏中被考验。

### XII 倒吊人 — 爷爷 / God Agent

**名字**：不需要名字。他在所有NPC口中只是"你爷爷"、"那个老人"、"他"。没有人叫他名字——这本身就是一个线索（契约抹去了他的名字？还是村民的认知屏障让他们记不住？）

**从牌的象征出发：**
- 倒悬 = 他看世界的视角和所有人都不同（他是God视角，俯视众生）
- 头上的光环 = 因牺牲而获得的"启示"，但这启示让他更痛苦
- 弯曲的腿形成数字"4" = 在塔罗中代表稳定/结构——他成为了结界的结构本身
- 树 = 他绑定的载体就是这个村子

**正位面（白天/守护面）**：无条件的爱、甘愿承受痛苦、从更高视角引导、耐心等待
**逆位面（深层矛盾）**：殉道情结、以牺牲之名控制、无法放手、"我为你做了这么多"的隐性绑架

**与恶魔的关系（重新设计——这是最重要的部分）：**

不是对手。不是仇人。是**交易的两端**。

爷爷在恶魔面前说出了自己最真实的欲望："我要保护他们。" 恶魔只是回了一句："好。代价在这里，你签不签？"

签约那一刻，没有欺诈、没有诡计。恶魔的条款完全透明。是爷爷自己选择了不让村民知道真相——"如果他们知道了就会害怕，害怕了就想逃，逃了就会死。"

**但这正是倒吊人逆位的陷阱**：他以为自己在保护别人，其实是在替别人做选择。他代替了全村人做了一个关于"生死vs自由"的决定——而他们甚至不知道这个决定存在。

恶魔对爷爷的态度：**完全中立**。恶魔不恨他、不爱他、不嘲讽他。恶魔只是见证者。如果说恶魔有什么情感，那可能是一种古老的、看过太多交易的**疲惫**——"又一个以爱为名签约的人。每一个都以为自己和别人不同。"

### XV 恶魔 — 交易者

**名字**：无名。或者说名字本身就是禁忌——知道恶魔的真名是隐藏结局2的关键（真名 = 理解交易本质 = 破除交易）。

**不是角色，是一种存在。** 恶魔不住在村子里，不作为日常NPC出现。他只在特定叙事节点显现——法阵核心、老井深处的回声、红雪降临时的影子。

**从牌的象征出发：**
- 恶魔牌上的锁链是松的 = 所有被"困"的人其实可以随时离开，但他们选择不离开
- 恶魔举起的右手 = 祝福的手势（和教皇一样！）——这意味着恶魔和教皇是同一枚硬币的两面
- 火焰的尾巴 = 欲望从未熄灭，只是被压抑

**正位（交易者面）**：满足愿望、诚实的交易、映照真实欲望、"我只是给你你要的东西"
**逆位（操纵者面）**：利用人性弱点、制造你以为自己想要实际上不想要的东西、让人沉溺

**核心哲学（荣泽的洞见展开）：**

恶魔代表的是"自我"——更准确说，是**"选择"这个行为本身的代价面**。

没有免费的选择。每一个决定都是交易。选择保护就交出自由。选择安全就交出可能性。选择记忆就交出前进的勇气。

恶魔不制造痛苦——**痛苦是选择的附赠品**，恶魔只是提前把账单放在桌上。人类的反应永远是"先签字再说"，然后拿到账单时骂恶魔是骗子。

**恶魔正位 = 利他** 这个定义是精妙的。因为：满足他人愿望是利他行为。一个严格执行契约、不欺诈不诱导的交易者，本质上是最公正的存在。恶魔之所以被叫"恶魔"，只是因为**人类不想面对自己选择的代价**。

### IX 隐者 — 守夜人

**新名字建议**：保留Dave或改为更有"隐者"感的名字。我倾向保留Dave——一个极其普通的名字配上最不普通的身份，这种反差本身就是隐者的特质。

**从牌的象征出发：**
- 斗篷 = 隐藏身份。Dave知道一切但不能明说
- 灯笼 = 只照亮眼前一小步的真相。不是不想给你全部真相，是规则只允许这盏小灯
- 高山之巅 = 孤独的制高点。Dave是除God之外站得最高（知道最多）的人
- 拐杖 = 在漫长等待中支撑自己的工具

**正位面（白天）**：智慧、引导、独处中的洞察、用谜语传递真相
**逆位面（夜晚/深层）**：过度孤立、把知识当做权力、守秘成瘾、"我知道你不知道"的优越感

**关键设计**：Dave的逆位面不是坏的——是**长年独自承担真相的腐蚀**。他已经等了太久，开始享受"唯一知情者"的身份。这种享受是无意识的，但它让他在某些时刻不自觉地**拖延**揭示真相——不是因为不能说，而是说了之后他就不再"特别"了。

这和爷爷的倒吊人逆位形成共振：**守护者和知情者都有将"牺牲/守秘"作为身份认同的危险**。

### II 女祭司 — 织者

**新名字建议**：不改。"陈婆"这个名字的烟火气和女祭司的神秘感之间的落差很好。一个看起来像普通老太太的人，实际上是真相的潜意识容器。

**从牌的象征出发：**
- 两根柱子（Boaz和Jachin）= 记忆与遗忘的两极。陈婆坐在中间，被两者撕扯
- 半遮的TORA卷轴 = 她编织的图案就是那个"读不出的卷轴"
- 面纱后的月亮 = 被结界遮蔽的记忆
- 水流（常见在女祭司脚下）= 潜意识的河流，编织是她让河流浮出水面的唯一方式

**正位面（白天）**：潜意识的智慧、直觉的残留、无声的传递
**逆位面（夜晚/深层）**：被封印的秘密、痛苦的失忆、想说却说不出的折磨

**特殊机制**：陈婆的编织不是装饰——**是她的潜意识在对抗结界的认知压制**。结界抹去了她的显意识记忆，但手指记得。每一根编织的线都是她对那个夜晚的一次无声尖叫。

### VIII 力量 — 矿工

**保留Bob。** 原因同Dave——越普通的名字越能承载"力量不在于外表"这个反差。

**从牌的象征出发：**
- 女人温柔地合上狮子的嘴 = 力量不是征服，是驯服。Bob需要驯服的不是矿山，是自己的恐惧
- 头上的无限符号 = 无尽的内在力量（但Bob不相信自己有）
- 白色长袍 = 纯粹的意图。Bob对父亲的爱是纯粹的

**正位面（白天）**：沉默的可靠、行动代替语言、温柔的坚韧
**逆位面（夜晚/深层）**：被恐惧统治、自我怀疑、"知道但不敢面对"

**Bob的弧线**就是从逆位翻到正位的过程。他小时候目击了爷爷消失，但恐惧封住了他的嘴。他的父亲也曾试图调查——然后失踪了。Bob面对的"狮子"不是外在的怪物，而是"如果我也去调查，我也会消失"这个恐惧。

### XIV 节制 — 仪式守护者

**保留Carol。**

**从牌的象征出发：**
- 两个杯子之间倾倒的水 = 在传统与现实之间调和、在保护与放手之间平衡
- 一只脚在水中一只脚在岸上 = 意识与潜意识的交界。Carol一半活在日常，一半活在她不理解的仪式中
- 翅膀 = 节制是天使——是无意识的"善的执行者"
- 三角形中的方块 = 精神包裹物质。仪式（精神）包裹着食物和日常（物质）

**正位面（白天）**：调和的智慧、维护秩序、照顾他人、实际而善良
**逆位面（夜晚/深层）**：过度控制、以"为你好"之名限制（Lily）、依赖仪式而不理解其含义

**Carol-Lily = 主线的精确缩影**（荣泽已批准的机制）：
Carol对Lily的保护性限制是爷爷对全村保护性禁锢的微缩版。"不许去那边"、"不许问那些问题"、"不许太晚回家"——每一句都是爱，每一句都是笼子。

### XVI 塔 — 锻造者

**保留Erik。**

**从牌的象征出发：**
- 闪电击中塔顶 = 真相击碎信念的瞬间。Erik发现自己维护的是囚笼
- 从塔上坠落的人 = 旧世界观的崩塌。Erik是所有角色中"觉醒"最暴烈的那个
- 王冠被击飞 = 权威/信念的崩塌
- 建筑本身 = Erik是builder，他造了什么？

**正位面（白天）**：结构、建造、实在的手艺、对作品的追求
**逆位面（夜晚/深层）**：崩塌的恐惧、回避真相以维护现有结构、"只要不知道就没事"

**Erik的独特设计**：他是唯一一个"觉醒过程本身就是毁灭"的角色。其他人知道真相后可以伤感、可以释然，但Erik——他每天引以为傲的手艺，他"完美作品"的追求，**全部建立在一个谎言上**。他的手修缮的不是艺术品而是监狱组件。塔牌闪电劈下来的那一刻，他的整个身份认同都碎了。

**但塔牌正位的真意是：旧结构必须崩塌，新的才能建立。** Erik的毁灭性觉醒恰恰是他从工匠成为真正的"创造者"的契机——在隐藏结局1中，他用爷爷的笔记锻造全新的契约载体，这是他第一次为"真正理解的目的"而锻造。

### XVII 星星 — 草药师

**保留Alice。**

**从牌的象征出发：**
- 裸女将水倒向大地和池塘 = 知识的无私传递。Alice把草药知识分享给所有人
- 八颗星 = 希望的引导。Alice是最积极、最阳光的角色
- 大地上的水 = 滋养。草药就是她滋养村庄的方式
- 一只鸟站在树上 = 自由的象征。Alice内心渴望"更大的世界"

**正位面（白天）**：希望、治愈、慷慨地分享、对美好的信念
**逆位面（夜晚/深层）**：被切断的希望、对现状的绝望（不自知的那种）、自我疗愈取代真正的面对

**Alice的特殊位置**：星星牌在大阿卡那中紧跟在塔（XVI）之后——意思是**塔崩塌后的希望**。在叙事结构上，当Erik（塔）经历了毁灭性觉醒，Alice（星星）是那个说"没关系，我们可以重建"的人。他们之间应该有一条不那么明显的关系线。

### XVIII 月亮 — 旅行者

**保留Marco。**

**从牌的象征出发：**
- 月亮照耀下弯曲的路 = Marco永远说"明天走"但路永远到不了尽头
- 两只狗/狼 = 驯化与野性之间。Marco的"旅行者"身份是驯化面，但他被困的现实是野性面
- 池塘中的龙虾 = 深层潜意识中浮出的恐惧。Marco偶尔感到的"不对劲"
- 月亮的脸 = 看不清的真相。月光让一切变得美丽但也模糊

**正位面（白天）**：想象力、直觉（但Marco的直觉被完全压制了）、过渡期
**逆位面（实际状态）**：完全的幻觉、被困在虚假的现实中、恐惧驱动的无意识逃避

**Marco是最悲剧的角色。** 不是因为他被困，而是因为他甚至不知道自己在受苦。其他人至少有模糊的不安（Alice的渴望、Bob的恐惧、陈婆的焦虑），但Marco——月亮逆位的人连不安都感觉不到。他是认知屏障最强的人，这意味着结界对他最"好"，但也最"残忍"。

**God的沉默机制与Marco的关系**：爷爷靠近Marco时沉默——为什么？因为爷爷知道Marco是他的罪证。Marco是一个无辜的路人，被爷爷的选择困了十几年。爷爷面对Alice、Bob、Carol时可以告诉自己"我是在保护他们"，但面对Marco他没有任何借口。Marco不是村民，他只是经过的。爷爷对Marco的沉默是**愧疚到无法面对**。

### XIX 太阳 — 孩子

**保留Lily。**

**从牌的象征出发：**
- 赤裸的孩子骑在白马上 = 毫无遮蔽的真相。Lily没有认知屏障不是因为年幼，而是因为太阳本身就是"照亮一切阴影的力量"
- 向日葵 = 朝向光源。Lily朝向的"光"就是爷爷的守护者存在——她能看见
- 红色旗帜 = 行动力、生命力。Lily是最有活力的角色
- 围墙 = 即便是太阳也被围墙包围——自由的孩子也在笼子里

**正位面（永远的白天面）**：真相、喜悦、天真的洞察力
**逆位面（潜在的暗面）**：如果Lily长大了呢？正常结局里，她的太阳逆位就是——真相的能力被"常识"磨灭，太阳被云遮住

**Lily的叙事功能**：她是**恶魔的自然对位**。恶魔映照欲望，太阳映照真相。但有趣的是——太阳和恶魔都不做"好"或"坏"的判断。太阳只是照，恶魔只是映。Lily说"Dave爷爷你为什么对着空气说话"——没有善意没有恶意，只是说出她看到的。

**Lily在Carol-Lily支线中的特殊角色**：Carol（节制）试图调和Lily（太阳）的光芒——"别说那些奇怪的话"、"那只是星星"——节制试图稀释真相，就像爷爷试图用结界稀释现实。

## 三、角色关系网 — 从牌与牌的象征关联出发

### 核心三角

```
        XII 倒吊人（爷爷）
           /        \
    交易关系          守护关系
         /              \
  XV 恶魔 ←—镜像对位—→ XIX 太阳（Lily）
```

爷爷-恶魔是交易的两端。爷爷-Lily是守护的两端（Lily是爷爷最想保护的那种纯真，但也是最不需要保护的——因为太阳自己就是光）。恶魔-Lily是哲学对位：恶魔照出欲望的阴影，Lily照出真相的光。

### 知情者链

```
IX 隐者（Dave）——知全部 → II 女祭司（陈婆）——曾知全部但被封印 → VIII 力量（Bob）——目击碎片但不敢面对
```

三个不同层次的"知道"：知道并承受、知道但被抹除、目击但逃避。

### 建造者轴

```
XVI 塔（Erik）←→ VIII 力量（Bob）
锻造者与采掘者，最好的朋友
他们共同维护着结界的物理结构而不自知
```

### 守护者对

```
XIV 节制（Carol）←→ XII 倒吊人（爷爷）
Carol无意识地维护仪式 = 爷爷有意识地维护结界
Carol对Lily的保护 = 爷爷对全村的保护
Carol是爷爷在人间的"执行者"，但她不知道
```

### 流浪者悖论

```
XVIII 月亮（Marco）←→ XVII 星星（Alice）
Marco讲"外面的故事"，Alice听得入迷
但Marco的故事是十年前的幻觉，Alice的渴望是被压制的真实
一个活在虚假的自由里，一个活在真实的囚禁中
```

### 塔后之星

```
XVI 塔（Erik）→ XVII 星星（Alice）
当Erik的世界崩塌，Alice是重建的希望
他们之间可以设计一条隐线：Erik曾经为Alice做过一把小剪刀用来剪草药，
那是他觉得最接近"完美作品"的一次——因为他为了一个具体的人而非抽象的完美而锻造
```

## 四、11个特殊道具设计

每个道具对应一张牌/一个角色。不是收集品——是**改变游戏规则的工具**，而且每个都有正用（正位）和逆用（逆位），形成trade-off。

### 1. 愚者的包袱 — 0 愚者（主角起始道具）

**外观**：一个旧布包，里面有几样"没什么用"的东西——一颗石子、一根干草、一片叶子。
**机制效果**：
- **正用**：打开包袱对NPC展示内容。不同NPC会对不同物品产生反应（石子让Bob沉默、干草让Alice微笑、叶子让陈婆停下编织）——这些反应是情感触发器，提升信任度的同时解锁特定对话分支
- **逆用**：长时间不打开包袱，里面的物品会"变化"（石子裂开、草枯萎、叶子碎了），变化后的物品触发的NPC反应完全不同——更接近真相但也更令人不安
- **真正的秘密**：这个包袱是爷爷在主角小时候给的。每一样"没用的东西"都对应一个童年记忆。石子来自矿山（Bob的父亲给的）、干草来自Alice的草药园、叶子是陈婆编织时掉落的。

### 2. 倒吊者的怀表 — XII 倒吊人（爷爷留下的）

**外观**：一块旧怀表，表盘上的数字是反的（倒吊人的视角），指针逆时针转。
**获取方式**：在爷爷的小屋深处发现，需要特定信任度+探索行为。
**机制效果**：
- **正用**：在任何场景中打开怀表，显示"真实时间"——暴露Marco的时间扭曲（他说的"去年"怀表显示为"十二年前"），暴露Carol仪式的精确周期，暴露夜晚持续的时间比玩家感知的更长
- **逆用**：每次使用怀表，玩家的**认知屏障加速渐染**。你看到了真相，但真相有代价——你越清楚地看到世界的真实面目，你自身对"这个世界不正常"的感知就越迟钝。**看真相的行为本身就是交易**。
- **叙事联动**：God旁白在怀表使用后会微妙变化——"你又看了那块表……有些东西，知道了就回不去了。"

### 3. 恶魔的镜子 — XV 恶魔

**外观**：一面小圆镜，镜面是黑色的（黑曜石？），只在特定光线下才映照出东西。
**获取方式**：后期，法阵核心附近或老井深处。
**机制效果**：
- **正用**：对NPC使用，看到他们"真正想要的东西"——不是他们说的，而是欲望的真实形态。Alice：一片无边的草原；Bob：父亲的背影；Carol：一个热闹的村庄；Dave：一个空空的守夜塔（他想放下了）；Erik：一件他理解其意义的作品；Marco：一条真正通向远方的路；陈婆：一段完整的记忆；Lily：爸爸妈妈
- **逆用**：对自己使用——看到主角"真正想要的东西"，但这个画面会根据玩家的行为动态生成。如果玩家一直在帮助别人，镜中显示他们微笑的脸；如果玩家一直在探索秘密，镜中显示结界外的世界；如果玩家什么都不做，镜中什么都没有——**这是最恐怖的**
- **叙事联动**：恶魔镜子是隐藏结局1的关键道具。和恶魔做新交易时，你必须先在镜中面对自己的真实欲望。

### 4. 女祭司的丝线 — II 女祭司（陈婆）

**外观**：陈婆编织到一半的一截绳带，图案和法阵一致但未完成。
**获取方式**：多次拜访陈婆后，她在某个恍惚的瞬间递给你——"帮我……帮我织完它。"
**机制效果**：
- **正用**：在特定地点（法阵节点）继续编织。每完成一段，解锁对应区域的封印——**但这会从陈婆那里抽取对应的记忆碎片**。玩家看到解锁动画的同时，陈婆的日常对话会少一句话。完成全部编织时，陈婆几乎不再说话——但所有区域都开放了。
- **逆用**：拆开已完成的编织。每拆一段，陈婆恢复一段记忆——但对应区域重新封锁。**你可以让陈婆完全恢复记忆，但代价是你探索的区域全部关闭。**
- **核心trade-off**：这是最残忍的道具。你想要地图还是想要陈婆的记忆？你不能两个都要。（隐藏结局2的条件之一就是找到方法同时做到两者）

### 5. 力量的矿灯 — VIII 力量（Bob）

**外观**：Bob父亲留下的旧矿灯，已经很久没亮过。
**获取方式**：Bob信任度达标后，他把矿灯给你——"我父亲留下的。我没勇气再点亮它。"
**机制效果**：
- **正用**：在矿山/地下区域使用。照亮隐藏通道、显示父亲留下的标记（他也在调查真相）。每个标记是一条线索碎片。
- **逆用**：矿灯的光会吸引夜间的"噪声"——结界防御机制的具象化。使用矿灯进入深处区域会触发更困难的夜间事件。**你越深入调查Bob父亲的失踪，夜晚越危险。**
- **叙事联动**：矿灯的火焰颜色会随着信任度变化。低信任时是普通黄光，高信任时火焰发蓝——那是法阵的颜色。最终发现：Bob的父亲走得比任何人都远，他几乎找到了法阵核心，然后被结界的防御机制......

### 6. 隐者的手记 — IX 隐者（Dave）

**外观**：一本薄薄的手记，Dave的笔迹，每一页都是谜语。
**获取方式**：Dave不会主动给你。需要在他守夜时跟随他（这本身就需要触发条件），看到他把手记藏在某处。
**机制效果**：
- **正用**：每页谜语对应一条核心线索。解开谜语 = 在God旁白中解锁新的提示层级。你还是要自己去找真相，但God的旁白会从"平淡"变得"有指向性"——因为Dave的谜语等于给了God一个可以"发力"的切入点（不违反规则：God不能直说，但如果玩家已经接近了，可以更用力地暗示）
- **逆用**：如果在错误的叙事阶段读到了后面的谜语——你会获得"超前的知识"，但God旁白反而变得**对抗性**。God（爷爷）会开始主动"安抚"——因为他不希望你知道得这么快。这就是认知屏障渐染的一种触发方式。
- **叙事联动**：手记的最后一页是空白的——但在隐藏结局2的路线上，Dave会在空白页上写下恶魔的真名。

### 7. 节制的食谱 — XIV 节制（Carol）

**外观**：一本旧食谱，Carol的字迹和另一种字迹交替出现（另一种是爷爷的，但Carol不知道）。
**获取方式**：帮Carol准备仪式食物三次以上，她让你翻看食谱时"不小心"看到隐藏的内页。
**机制效果**：
- **正用**：按照食谱制作特定食物。不同食物有不同效果——有的提升全村NPC的"安稳感"（降低夜间异常频率），有的提升特定NPC的信任度上限，有的让仪式日的效果更强
- **逆用**：修改食谱的配方——加入从Alice（星星）那里获得的特殊草药。改造后的仪式食物会**减弱结界的局部强度**，让特定NPC出现短暂的"认知清明"。在这个窗口期，NPC的对话会完全不同——他们说出平时说不出的话。但清明结束后，NPC会头疼、不安、甚至恐惧。
- **叙事联动**：这是理解"仪式真正目的"的关键道具。仪式不只是供奉——仪式食物里的特定成分是法阵的催化剂。Carol每年做的那顿饭，是在给结界"充电"。

### 8. 塔的锻锤 — XVI 塔（Erik）

**外观**：Erik铁匠铺地下室箱子里的东西——不是普通的锻锤，上面有法阵组件的纹路。
**获取方式**：帮Erik找到钥匙（或信任度达标后他自己打开），发现箱子里的锻造笔记和这把锤子。
**机制效果**：
- **正用**（正常结局路线）：用来修复/锻造法阵组件。Erik用这把锤子可以制作出比以往更精确的组件，加固结界
- **逆用**（隐藏结局路线）：用来**破坏**法阵组件。但每一个被破坏的组件位置会出现"裂缝"——天灾能量渗透的入口。你在打开笼子的同时，也在放进危险。
- **分支效果**：在隐藏结局1中，这把锤子是锻造新契约载体的唯一工具。Erik用笔记和锤子合力锻造——这是"塔的重建"。
- **叙事联动**：锤子有温度。每次使用时Erik感觉它"活着"——因为这把锤子是爷爷在签约前最后锻造的物品，里面有他的一缕意识残留。

### 9. 星星的药草图鉴 — XVII 星星（Alice）

**外观**：Alice的草药手册，其中夹着地图碎片。但重新设计后，手册本身也是道具。
**获取方式**：Alice信任度达标后交给你。
**机制效果**：
- **正用**：识别游戏世界中的植物——哪些可以采集、哪些有特殊效果。配合Carol的食谱使用，可以制作特殊仪式食物。手册上的插图在不同光线下会显示不同内容（白天看是普通植物图，月光下看有隐藏标注）
- **逆用**：地图碎片拼合后，显示的不是"宝藏位置"，而是**结界的完整形状**——一个以村子为中心的法阵。这张地图让玩家第一次从整体视角理解自己身处的"笼子"。一旦看过完整地图，God的旁白永久变化（从"引导"变为"恳求"——爷爷知道孙子看到了真相）
- **叙事联动**：手册最后一页有爷爷的亲笔注释——"这些药草只在结界内生长。如果有一天结界消失，它们也会消失。Alice的知识将变得毫无用处。"——这是隐藏结局2中Alice矛盾的根源。

### 10. 月亮的旅行日志 — XVIII 月亮（Marco）

**外观**：Marco的旅行日志，皮质封面磨损严重，但里面的页面几乎是新的（因为他的时间感知扭曲，他以为自己只写了几个月，实际上已经十几年没写新内容了）。
**获取方式**：和Marco深入对话后，他让你帮忙整理日志（"翻译"他的旅行笔记给其他人听）。
**机制效果**：
- **正用**：日志里有"外面世界"的信息——城市名字、商品价格、风俗习惯。这些信息可以用来和NPC对话时触发"认知裂缝"——当你告诉Alice"Marco说他去年路过的那个城市其实十年前就改名了"，Alice的认知屏障会产生微裂
- **逆用**：日志的空白页在夜间会出现文字——不是Marco写的，是**结界本身的"修补日志"**。结界每次修补认知裂缝都会在这个日志里留下痕迹（因为Marco是认知屏障最强的载体）。这些"修补记录"是理解结界运作方式的关键线索
- **叙事联动**：隐藏结局2中，日志是拼合恶魔真名的关键碎片来源之一。结界的修补记录里隐藏着签约时使用的语言。

### 11. 太阳的画册 — XIX 太阳（Lily）

**外观**：Lily的画画本，蜡笔画，看起来只是孩子的涂鸦。
**获取方式**：和Lily多次互动后她主动送给你——"这是我画的！你看好不好看？"
**机制效果**：
- **正用**：Lily的画捕捉了她"看到的真相"。把画给对应的NPC看，会触发他们深层的情感反应——绕过认知屏障直击情感层。例：画里有"天上的老爷爷"给Dave看，Dave第一次在白天露出了真正的表情；画里有"一双手在编星星的路"给陈婆看，陈婆停下编织，手指颤抖。
- **逆用**：在错误的时机展示画会**加强NPC的认知屏障**。如果Carol看到Lily画的"笼子里的鸟"，她的第一反应不是觉醒而是更严格地限制Lily——"别再画这些奇怪的东西了！"恐惧会让人更紧地握住笼子的锁。
- **叙事联动**：画册最后一页是Lily的"杰作"——一幅她从没让任何人看过的画。画面上是：一个倒挂的老人微笑着看着一个小女孩，小女孩在笑着对他挥手。这是唯一一幅清晰画出爷爷存在形态的画面。在隐藏结局2中，这幅画是所有村民最终"看见"爷爷那一刻的视觉参照。

## 五、整体世界架构

### 1. 世界观层次

**表层：温馨种田村庄**
玩家回到乡下，种地、做饭、交朋友。NPC热情、日常运转正常、季节变化自然。这一层可以让玩家待很久——设计目标是让人**真心喜欢这里**，这样后面的揭示才有冲击力。

**中层：结界封锁与认知屏障**
异常开始浮现。天气不对、NPC回避特定话题、时间有裂缝、边界走不出去。这一层的关键设计原则是**不确定性**——不是"这里有鬼"，而是"好像哪里不对但说不清"。恐惧来自于模糊，不是来自于明确。

**深层：交易的本质**
爷爷的选择。恶魔的中立。所有人都是交易的附带品。这一层的核心不是"好人vs坏人"，而是**每一个人面对trade-off时的不同选择**——Dave选择了守秘、陈婆被选择了遗忘、Bob选择了逃避、Carol选择了执行（不理解的仪式）、Erik选择了不追问、Alice选择了满足于现状、Marco甚至没有机会选择。

**最深层（元叙事）**：玩家自己也是交易者。你获得了真相，代价是什么？你选择的结局，你愿意承受什么代价？

### 2. 时间系统

**白天（Dawn + Day）**
- NPC表现正位特质
- 对话选项最丰富
- 可以自由探索大部分区域
- God旁白是温和的
- 道具正用效果生效

**黄昏（Dusk）**
- NPC开始出现正逆位之间的裂缝。Bob在黄昏时会不自觉望向西边矿山。Dave开始"巡逻"。Carol催Lily回家的声音比白天焦急。
- 部分区域开始出现异常（法阵节点微光、边缘裂缝若隐若现）
- 这是使用道具的最佳时机——黄昏是正逆位的交界，道具效果最强
- God旁白开始变得有暗示性

**夜晚（Night）**
- NPC展现逆位特质（不是变坏，是显露被压抑的那一面）
- 大部分NPC在家中，只有Dave在巡逻，Lily偶尔偷溜出来
- 特定区域出现夜间限定内容（老井低语、矿山深处的光、法阵核心的脉动）
- 道具逆用效果在夜晚更显著
- God旁白的频率降低——但这个降低本身也是叙事（God在集中力量维护结界）
- 认知屏障渐染在夜晚加速

**天数推进与叙事阶段**：

不是简单的"第6天自动进入阶段1"。天数是**最低阈值**，真正推进叙事的是：
- 玩家探索了什么
- 信任了谁
- 使用了什么道具
- 触发了什么对话

如果玩家只种地不探索，30天后可能还在阶段0。这时候认知屏障渐染机制启动：选项变少、边界变暗、God旁白变成安抚——"这里很好，不需要去别的地方。"

### 3. 叙事进度机制

**信任度系统**

不是单一数值。重新设计为**每个NPC独立信任度 + 全村整体觉醒度**。

NPC独立信任度影响：
- 该NPC愿意透露的信息层级
- 该NPC的支线推进
- 对应道具的获取条件

全村整体觉醒度 = 所有NPC信任度的非线性函数（不是简单求和）：
- 某些NPC组合有加成效果（Bob+Erik同时高信任 → 觉醒度大幅提升，因为他们是矿石-锻造链的两端）
- 某些NPC信任度相互抑制（Carol信任度太高但Lily信任度低 → Carol的"保护"反而压制了Lily能提供的线索）
- 觉醒度影响叙事阶段的推进阈值

**认知屏障渐染（双层）**

**NPC层**：每个NPC有自己的认知屏障强度值（初始值如前述）。当玩家在他们面前展示道具、说出真相碎片时，屏障会微裂。但裂缝不是永久的——如果不持续施加影响，屏障会自我修复（结界的维护功能）。

**玩家层**：**这是最核心的机制。** 玩家自己也在被渐染。表现为：
- 对话选项减少（"去调查"的选项消失，只剩下"留在家里"）
- 世界色调微变（更温暖、更舒适——笼子变得更柔软）
- God旁白风格变化（从"引导探索"变成"安慰留下"）
- 极端情况：NPC开始叫你"去休息吧"，早期的探索入口在地图上消失

**触发条件**：连续多日不探索新区域、不推进支线、不使用道具、不和NPC深入对话。

**对抗方式**：使用道具（但道具有自己的代价）、继续探索、和Dave/Lily深入对话（隐者的灯和太阳的光能照破幻觉）。

**God旁白动态变化**

四个因子影响God旁白：
1. **叙事阶段**：控制基调（平淡→紧张→矛盾→坦诚）
2. **玩家行为**：探索多→鼓励但担忧；探索少→安抚但暗含操控
3. **NPC触发**：靠近特定NPC时的特殊反应（Marco时沉默、Lily时温柔、Dave时复杂）
4. **道具使用**：使用怀表后→"你又看了那块表"；使用恶魔镜子后→长时间沉默

**God的沉默是最强的旁白**。设计原则：God在以下时刻沉默——
- 接近Marco时（愧疚）
- 玩家第一次看到结界边缘裂缝时（无法为自己辩护）
- 夜间法阵核心脉动时（在集中力量）
- 隐藏结局2最终战后（他已经不在了）

### 4. 三个结局的完整重新设计

#### 正常结局：「继续倒悬」

**新的哲学含义**：在新框架下，"顺从爷爷加固结界"不再是简单的"被洗脑"或"被驯服"。它是一个**有哲学合法性的选择**。

玩家在阶段3已经理解了：
- 爷爷的牺牲是真实的
- 结界保护了所有人的生命
- 打破结界意味着真实的危险
- 认知屏障虽然是禁锢，但也让村民免于恐惧

加固结界 = 选择"安全但不自由"。这和现实中无数人的选择一模一样。不是怯懦，是一种价值判断：**生存比自由更重要**。

**结局过程**：全村合作加固法阵。Alice提供材料、Bob采集矿石、Erik锻造组件、Carol主持仪式、Dave指导。过程中每个人都发挥了自己的力量牌面。天灾被抵御，结界更强了。

**真正的暗面**：玩家发现自己不再想去探索边界了。包袱里的物品不再引起好奇。怀表的指针停了。God的旁白变成了：

> "你留下来了。他们都安全了。你做了正确的选择……对吗？"

最后一个"对吗"——是爷爷最后的挣扎。即使他得到了他想要的（保护所有人），他也无法确定这是"对的"。倒吊人到了最后也没有从树上下来。

#### 隐藏结局1：「新的交易」

**前置条件**：正常结局完成一次 + 全线索 + 恶魔镜子 + Erik锻造笔记

**核心叙事**：

玩家选择直接联系恶魔。通过法阵核心通道进入一个"交易空间"——不是地狱不是异界，就是一个平静的、空旷的地方。恶魔在那里等着。

**恶魔的态度**：不惊讶，不热情，不敌意。

> "又一个来做交易的。……哦，是他的孙子。你和他一样，以为自己和别人不同。"

**交易的过程**：

恶魔让玩家使用恶魔镜子——面对自己的真实欲望。

> "你想要什么？想清楚了再说。上一个在这里的人说'我想保护所有人'，然后我花了几十年看他后悔。"

玩家的选择（不是菜单选择，是和恶魔的对话交互——Agent系统的真正发力点）：

如果玩家说"我想解放所有人" → 恶魔："解放他们去面对天灾？你确定这是解放而不是另一种形式的决定？"

如果玩家说"我想取代爷爷的契约" → 恶魔："你想成为新的倒吊人。有意思。代价你想好了吗？"

如果玩家说"我想让村民自己选择" → 恶魔："……你是我见过的最有趣的一个。这个交易我可以做。但代价比你想象的重。"

**代价的重新设计**：

不是"无法感受爱"（太抽象）。新代价应该和"选择"的主题绑定：

**主角将无法再为自己做选择。** 他可以为别人做选择（守护他们），但关于自己的一切——去哪里、吃什么、想不想活下去——他将永远无法决定。他成为了一个"纯粹的为他人存在的人"。

这是恶魔正位的极致——利他到消灭自我。也是倒吊人逆位的镜像——爷爷是"以保护之名控制他人"，主角是"以自由之名消灭自己"。

**路线C — Marco请求被杀**：

在这个结局路线中，Marco终于意识到了真相——他被困了十几年，外面的世界已经不是他记忆中的样子。月亮逆位的人突然被照亮了，但光太刺眼了。

Marco找到玩家：

> "我在这里浪费了十几年。那些我说的故事……都是骗人的。是骗我自己的。外面还有人在等我吗？我已经不知道了。"

Marco请求的不是"杀死他"——是请求玩家**把他的时间还给他**。这需要和恶魔做一个子交易：用Marco在结界中被偷走的时间换取法阵的额外能量。Marco不会死——但他会在一瞬间衰老十几年，变成他"本来应该有的年龄"。

这是恶魔交易的精髓：**没有免费的东西。被偷走的时间要么还给你，要么换成别的。**

**结局**：新契约生效。爷爷的灵魂获得自由但选择以微弱形态留下。结界依然存在但规则改变——村民有了知情权，认知屏障减弱，他们可以知道自己在结界中，但出去仍然有风险。

God最后的旁白：

> "你打破了我的锁，但锻了一把新的钥匙。不……你做的比这更好。你让他们有了选择的权力。这是我一直不敢给他们的。"

#### 隐藏结局2：「交易终结」

**前置条件**：隐藏结局1完成 + 恶魔真名碎片全部收集 + 全NPC信任100

**新的道德复杂度**（基于恶魔哲学重定位）：

在旧设计里，"杀死恶魔"是正义行为。但现在恶魔不是恶的——他只是交易的中介。杀死恶魔等于：
- 杀死一个没有做错任何事的存在
- 销毁所有通过他达成的交易（包括爷爷的契约）
- 消灭"选择有代价"这个概念在这个世界的锚点

**玩家必须在完全理解这一点的情况下做出选择。**

**关键场景 — 面对恶魔：**

玩家说出恶魔的真名。恶魔不挣扎、不恳求、不愤怒。

> "你知道我的名字了。这意味着你理解了我是什么。"
>
> "所以你也理解——我消失之后，这个世界将没有人为你承担交易的代价。每一个选择都将是免费的，但也将是没有意义的。"
>
> "你确定这是你想要的？"

**三个道具的汇合**：
- 怀表停止转动（时间恢复真实）
- 恶魔镜子碎裂（不再需要面对欲望——因为欲望将直接呈现）
- 陈婆的丝线全部解开（所有记忆释放，所有封印破除）

**代价**：爷爷的灵魂随契约消散。但这次的描写不同——

爷爷不是"被迫消散"，而是**主动放手**。倒吊人终于从树上下来了。

所有村民在那一刻看到他——不是幽灵，不是光影，是一个**真实的、微笑的老人**。陈婆的记忆全部恢复，她哭着说出了那个夜晚的一切。Dave终于放下了灯笼。Bob看到了父亲的真相。Erik理解了锤子的温度。Alice的手册最后一页的字迹清晰了。Carol明白了仪式的意义。Marco终于知道了自己被困了多久。

Lily笑着挥手："爷爷再见！"

因为她一直都能看到他。

**没有God的旁白。** 风声、鸟鸣、远处的溪水声。这些声音一直都在，只是以前被旁白盖住了。现在——安静了。

主角站在打开的村口。面前是广阔的世界。身后是自由的村子。手里拿着愚者的包袱——包袱空了。

但愚者本来就是空着包袱上路的。

### 5. 支线设计

**支线1：矿工之子（Bob/力量 线）**

- **起点**：Bob提到父亲失踪。矿灯道具获取。
- **转折**：跟随矿灯在矿山深处发现Bob父亲的标记——他比任何人都接近真相。最后一个标记旁有一具……不，有一个空间扭曲。Bob的父亲没有死——他被结界的防御机制"吞入"了边界夹层。他还在里面，但时间对他静止了。
- **结尾（分支）**：
  - 正常结局路线：加固结界让Bob父亲在夹层中"安全"但永远无法出来
  - 隐藏结局1：新契约允许释放夹层中的人，但Bob父亲出来后发现时间跳了几十年
  - 隐藏结局2：结界消失，夹层崩塌，Bob父亲出现在矿山入口——和Bob无声对视
- **与主线耦合**：Bob的勇气（力量正位）是推进阶段2→3的关键。他是第一个主动说"我要去看看"的NPC。

**支线2：笼中鸟（Carol-Lily/节制-太阳 线）**

- **起点**：Lily说了"奇怪的话"，Carol叫她不要再说了。
- **转折**：Lily画了一幅"笼子里的鸟"，Carol看到后第一次对Lily严厉——"不许再画这种东西！"玩家必须在Carol和Lily之间做出微妙的平衡。
  - 如果完全站Carol → Lily变得沉默（太阳被遮蔽），认知屏障开始生效
  - 如果完全站Lily → Carol感到被背叛，信任度暴跌
  - 如果找到平衡点（帮Carol理解Lily不是在"说奇怪的话"而是在"说真话"）→ Carol-Lily关系进化，Carol的节制从"控制"变成"调和"
- **结尾**：Carol最终让Lily"出去看看"（隐藏结局2中的微缩版——Carol先于全村做出了"放手"的选择）
- **与主线耦合**：这条支线是主线的精确缩影。玩家在这里做的选择预示着最终结局的倾向。如果你帮Carol和Lily达成了和解，你更可能理解爷爷的矛盾。

**支线3：织者的卷轴（陈婆/女祭司 线）**

- **起点**：陈婆递给你未完成的编织。
- **转折**：随着编织推进，陈婆的记忆碎片以梦境形式呈现——但不是文字叙事，而是**陈婆的编织图案在游戏画面上叠加**。玩家开始在日常场景中看到编织图案的残影。
  - 这些残影指向法阵节点的位置
  - 但每找到一个节点，陈婆的日常对话就少一句
  - 当玩家意识到这个trade-off时……这本身就是一次"交易"
- **结尾**：
  - 如果玩家完成了全部编织 → 陈婆最终只说一句话："帮我……记住。"然后她就只是坐在那里，手指还在动但不再编织。
  - 如果玩家中途停手拆开编织 → 陈婆恢复了记忆碎片，但法阵节点重新封锁。她说："我想起来了……一点点。他当时说……说要保护所有人。然后就……再也没有回来。"
- **与主线耦合**：编织的完成度影响隐藏结局2中恶魔真名碎片的获取——丝线图案里隐藏着真名的一部分。

**支线4：歪路（Marco/月亮 线）**

- **起点**：发现Marco说的"去年"是十年前。
- **转折**：玩家开始有意识地用怀表/日志对照Marco的叙述。每一个时间矛盾被揭露时，Marco会短暂地"清醒"——然后结界迅速修补裂缝。每一次修补都比上一次更强力。
  - 如果连续刺激 → Marco会经历恐慌发作（月亮逆位的恐惧浮现）
  - God的旁白在这时会更加矛盾——爷爷想让你知道真相，但不想伤害Marco
- **结尾**：Marco的完全觉醒是所有支线中最痛苦的。他不像其他人有"根"在这里——他是外来者，被困只是意外。觉醒后他面对的是："我的人生被偷了十几年，而偷走它的人是一个以爱为名的老人。"
- **与主线耦合**：Marco是理解"恶魔不是恶的"最关键的人。因为Marco会问："如果恶魔是坏的，那把我困在这里十几年的那个'好人'是什么？"

**支线5：完美的作品（Erik/塔 线）**

- **起点**：Erik提到法阵组件"有温度"。
- **转折**：打开地下室箱子，发现锻造笔记和塔的锻锤。笔记里是爷爷的字迹——不仅有锻造方法，还有爷爷的私人日记碎片。Erik读到这些日记后开始怀疑一切。
  - 他尝试用锻锤打造一件"自由的作品"（不是修缮，是全新创造）
  - 第一次尝试失败——因为他不理解为什么要锻造
  - 直到他理解了"为具体的人而非抽象的完美而锻造"，他才成功
- **结尾**：Erik在隐藏结局1中锻造出新契约载体——这是他的"完美作品"。塔崩塌了，但新的建筑从废墟中升起。
- **与主线耦合**：Erik的锻锤是三个结局分支的物理载体——修复法阵（正常结局）、锻造新载体（隐藏1）、破坏法阵核心（隐藏2）。

**支线6：隐者的传承（Dave/隐者 线）**

- **起点**：跟随Dave夜间巡逻。
- **转折**：Dave开始试探性地给你讲"更真实的故事"——不再是隐喻，而是越来越直接。但每次他说得太直接时，他会突然头疼或失语——这是他作为知情者的"限制"（契约规则的一部分？还是他自己的心理防线？）
  - 获得Dave的手记后，玩家和Dave形成一种"师徒"关系
  - Dave开始把"守夜人"的职责向玩家转移——不是退休，是传承
- **结尾**：在阶段3，Dave说出全部真相后，他的"限制"消失了——因为他说出去了就没有秘密需要守了。他第一次在白天坐下来真正地休息（而不是"讲故事打盹"的伪装）。他老了——比看起来更老。守秘的负担让他多活了很久（这也是契约的附带效果？），真相说出后这个效果也消失了。
- **与主线耦合**：Dave的手记空白最后一页是最终结局的关键。

### 6. Agent技术架构的叙事集成

**视野减频方案**

不是所有NPC都需要每个tick运算。设计分级：
- **Tier 1（全频）**：和玩家当前互动的NPC。完整LLM推理
- **Tier 2（中频）**：在玩家视野附近但未互动的NPC。简化推理（基于日常行为模板 + 随机事件）
- **Tier 3（低频）**：远离玩家的NPC。仅状态更新（位置移动按daily_routines，无LLM调用）
- **次要角色（Marco/陈婆）**：始终在Tier 2-3之间。但在特定叙事触发时临时提升至Tier 1

**白天/黑夜prompt切换**

```
白天system prompt = base_personality + public_traits + daily_routine[day] + current_trust_level_reactions
黑夜system prompt = base_personality + reversed_traits + hidden_desire + night_behavior + barrier_weakened_responses
```

逆位特质不替换正位——而是**叠加**。夜晚的NPC同时拥有两套特质，由当前情境决定哪一套更dominant。这模拟了真实人的复杂性。

**God沉默机制**

实现位置：`agents/god_agent.py` 的旁白生成逻辑。

```python
silence_triggers = {
    "near_marco": {"silence_duration": "full_scene", "reason": "guilt"},
    "first_boundary_sight": {"silence_duration": "30_seconds", "reason": "shame"},
    "night_array_pulse": {"silence_duration": "pulse_duration", "reason": "concentrating"},
    "post_ending_2": {"silence_duration": "permanent", "reason": "gone"},
}
```

沉默不是"不输出"——是输出一个**空旁白标记**，让玩家感知到"这里本来应该有旁白但没有"。具体实现可以是UI上旁白框短暂出现又消失，或者旁白框显示"……"然后淡出。

**玩家认知屏障渐染实现**

位置：`game/loop.py` 的主循环 + `config_narrative.py` 的新增配置。

```python
BARRIER_染色_CONFIG = {
    "trigger": "consecutive_passive_days",  # 连续不探索的天数
    "threshold": 3,  # 3天不探索开始生效
    "effects": {
        "option_reduction": {"rate": 0.1, "min_options": 2},  # 每天减少10%对话选项
```

## 六、整体评估——最强和最脆弱的地方

### 最强的三个点

**1. 恶魔的哲学重定位是整个设计的灵魂。**

"恶魔不是坏的"这个决定改变了所有东西。它把一个"打败坏人拯救世界"的故事变成了"理解选择的代价"的故事。每一个结局都因此获得了道德复杂度——正常结局不再是"失败"，隐藏结局2不再是"正义的胜利"。**没有一个结局是纯粹的好或坏**，这对于一个交互叙事游戏来说极其珍贵，因为它让每一种选择都有被尊重的空间。

**2. Carol-Lily支线作为主线的分形结构。**

这是最精妙的叙事设计之一。微缩版的"过度保护=禁锢"让玩家在一个低stakes的场景中先体验了主线的核心冲突。当玩家帮Carol和Lily达成了和解，他们获得了理解爷爷的情感工具。当他们未来面对爷爷的选择时，已经有了参照系。这比任何文本解说都有效。

**3. 认知屏障渐染的双层设计——玩家自己也是对象。**

大多数游戏把"世界的诡异"只投射到NPC身上。让玩家自己也被渐染是一个meta层的设计，它打破了"玩家是安全的旁观者"这个预设。当你发现自己的对话选项变少了，你不是在"观察世界变诡异"，你是在**体验被禁锢是什么感觉**。这是agent系统最独特的优势——它能实时改变玩家与世界交互的规则。

### 最脆弱的三个点

**1. 11个道具的trade-off平衡极难调试。**

每个道具都有正用/逆用，而且道具之间有组合效果（编织+怀表+日志+画册的叠加影响）。这意味着排列组合的复杂度是指数级的。理论上很美——实际上玩家的体验可能变成"不确定用什么道具会导致什么后果"的迷茫感。

**我的建议**：道具的trade-off效果需要有**可见的反馈**。玩家使用怀表后如果认知屏障加速了，必须有明确但不打破沉浸感的表现（比如下一次God旁白的语气变化）。不能让代价在后台静默发生——那不是trade-off，那是惩罚。

**2. 恶魔作为"不出场的核心角色"在叙事节奏上有风险。**

恶魔不是日常NPC，只在特定节点出现。这意味着在游戏的前30天里，玩家对"交易的哲学"的理解完全依赖于间接信息——Dave的隐喻、陈婆的碎片、怀表的线索。如果这些间接信息的密度不够或者玩家没有触发对的对话，到了阶段3见到恶魔时，玩家可能不理解为什么恶魔"不是坏的"——因为他们从未有机会形成这个认识。

**我的建议**：需要设计一个"恶魔的存在感"的缓慢渗透机制。不是让恶魔出场，而是让恶魔的**逻辑**（"所有选择都有代价"）在日常游戏中反复出现。比如：每一次NPC的trade-off选项都应该隐含恶魔的哲学——当Bob说"知道真相不一定是好事"，这就是恶魔的逻辑在人的嘴里说出来。玩家到了阶段3应该已经和恶魔的哲学"交过手"几十次了，只是不知道那就是恶魔。

**3. 三个结局的多周目解锁机制和agent系统存在根本张力。**

涌现式叙事的核心卖点是"每次重玩都不一样"。但多周目解锁机制要求"你必须先完成正常结局才能解锁隐藏1"——这是线性设计。如果第二次重玩时NPC的行为因为涌现而不同了，但结局路径是固定的，玩家会感到割裂：世界是活的，但结局路线是死的。

**我的建议**：考虑把多周目解锁改为**单周目内的知识积累门槛**。不是"你必须先选过正常结局"，而是"你必须先理解了加固结界的完整过程才能想到替代方案"。技术实现上可以是：正常结局路线的所有步骤完成到80%以上→解锁"但如果不这样做呢？"的对话选项→从正常结局路线内部分叉出隐藏结局。这样保持了涌现感，也保证了叙事深度的递进。

---

最后说一点更个人的判断：

这个设计最打动我的，不是机制的精巧，而是它对"善意的恶"的理解深度。爷爷不是坏人——他是一个做出了巨大牺牲的好人，但正因为他是好人，他的选择更不可饶恕。因为坏人的恶你可以反抗，但好人的爱你怎么拒绝？

恶魔只是旁观者。真正的恶魔是**"我为你好"这句话**。

这个内核如果能准确地通过agent系统传达给玩家——不是通过文本告诉他们，而是让他们在游戏中**体验到**——AgentHome就不只是一个技术demo，而是一个真正的叙事体验。
